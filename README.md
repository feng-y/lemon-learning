# Lemon Learning

这是一个面向 Lemon 的长期家庭学习系统。对象是正在成长的人，因此这里不追求“流程越复杂越好”，而是追求：**规则稳定、方法清楚、逐步放手、能长期迭代。**

## 当前定位

Lemon 当前 10 岁，处于 **2026 年暑假：三年级结束、即将进入四年级** 的过渡阶段。

当前优先级不是拔高或提前学完整个四年级，而是利用 **三年级复习 + 四年级适量预习** 作为诊断载体，暴露并修复会在四年级继续放大的问题：

- 能自己进入任务，而不是每一步都等父母安排；
- 遇到困难不立即心理退出、乱填或逃避；
- 书写遵循正确笔顺，提交内容清晰可识别；
- 数学能完整读题、定位信息、建立关系、计算并检查；
- 语文从字词理解逐步生长到阅读和表达；
- 英语以听、读、说、写和复习构成稳定语言闭环；
- 父母从“情绪施压者”转为规则维护者和学习教练。

## 当前三升四策略

当前阶段采用“双轨”：

> **复习旧内容确认基础是否稳定 + 预习新内容提前暴露下一阶段问题**

但两者都不是目的。

真正要解决的是：

- 复习后仍反复出现的基础漏洞；
- 预习新内容时暴露出的读题、启动、理解、检查和表达问题；
- 难度提升后出现的心理退出和任务质量坍塌；
- 需要父母持续实时调度才能完成的学习流程。

因此暑假结束时，优先看能力和执行质量有没有变化，而不是四年级教材提前学了多少。

## 核心原则

1. **不会可以，不认真不可以。**
2. **任务可以暂停，不能靠发火、拖延或乱写让任务消失。**
3. **先建立外部规则，再逐步转成自我管理。**
4. **学习时不听故事；听故事作为独立活动存在。**
5. **减少无边界、低成本、高刺激的即时娱乐，但不剥夺运动、陪伴、正常休息和基本快乐。**
6. **普通问题轻量处理，反复错误才进入深度修复。**
7. **Manual 维护标准，AI 按需扩展内容，孩子负责真实执行。**
8. **所有方法都服务于下一步能力，不提前训练远超当前阶段的复杂技巧。**
9. **长期原则、当前画像、长期进度、教学方法和每日执行必须分层维护。**
10. **任何新增规则都应有明确问题、验证方式和退出条件。**

## 当前策略

先冻结 v1 的责任边界和最小方法。后续修改优先来自真实执行中的反复问题和长期进度数据，而不是每次冲突后新增规则。

仓库长期维护的是**标准、方法、阶段画像和能力变化轨迹**，不是复制教材、堆积任务或持续扩张流程。
