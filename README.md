# Lemon Learning

面向 Lemon 的长期学习与能力成长工程。

长期目标不是提前中考化，也不是建立一个越来越强的家长管理系统，而是：

> **借助家长、家教和 AI 提供恰到好处的教学与支架，减少低效和无效努力，帮助 Lemon 建立扎实基础、有效学习方法、学习耐力和正向自我效能，逐步形成自主学习能力，使他既能适应海淀长期高强度的学习环境，又不因过度竞争、过度控制和持续失败体验损伤学习兴趣。**

当前：2026 暑假，三年级结束、即将进入四年级。

- 语文：人教版四上
- 数学：北师大版四上
- 英语：PEP 四上 + Power Up 3

## 平时怎么用

日常只从 [`active/CURRENT.md`](active/CURRENT.md) 开始。

```text
这月 / 这周选少数重点
        ↓
直接教学与训练
        ↓
按结果检查是否掌握
        ↓
给出具体正反馈
        ↓
记录高信号变化
        ↓
调整下一周
```

需要时再下钻：

- [`active/curriculum/`](active/curriculum/README.md)：查三科教材地图、单元重点和掌握标准；
- [`core/`](core/README.md)：查跨阶段复用的教学方法和原则；
- [`templates/`](templates/README.md)：Lemon 尚未内化某个动作时使用的临时支架；
- [`active/problems.md`](active/problems.md)：只跟进持续跨周、明确阻碍教学的问题；
- [`history/`](history/README.md)：回看过去阶段。

## 长期判断

小学后三年优先积累的不是提前学初中内容，而是未来进入高强度学习环境所需的承载能力：

> **基础扎实与逐渐自动化 + 学习效率 + 学习耐力 + 挑战耐受 + 学习兴趣 + 逐步自主。**

家长、家教和 AI 的角色都是临时支架。长期成功的标志不是成人介入越来越精细，而是 Lemon 越来越能回答：

> **我要学什么？我哪里不会？我可以用什么方法？我怎么知道自己真的学会了？**

## 执行约束

1. 当前重点最多 5 个；
2. 每周主任务最多 3 个，原则上每科最多 1 个；
3. Curriculum 可以完整，但不等于全部同时执行；
4. 训练必须有明确目标和检查方式，不用题量代替掌握；
5. Problems 同时最多 3 个，没有持续影响正常教学就不升级；
6. Progress 只记录真正改变下一步教学判断的证据；
7. 成人负责减少无效摩擦，但不替代 Lemon 必要的思考、提取、练习和坚持；
8. 每月反向检查一次：这套系统是否让 Lemon 更会学、更独立、更有信心，还是只是更依赖成人、更疲惫、更抗拒。

> **这个 repo 的价值不在于文档完整，而在于帮助 Lemon 用更高质量的学习获得真实进步，并让外部支架最终逐渐退出。**
