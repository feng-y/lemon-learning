# Lemon Learning

面向 Lemon 的长期学习改进工程。

目标是持续回答三件事：

1. Lemon 现在最需要解决什么？
2. 这一阶段怎样训练和验证？
3. 一段时间后，是否真的变好了？

当前阶段：**2026 年暑假，三年级结束、即将进入四年级。**

## 从这里开始

日常只需要进入 [`active/`](active/README.md)。

```text
active/      现在：状态、问题、进度、临时记录
core/        长期：稳定原则与教学方法
templates/   执行：Lemon 可直接使用的轻量模板
history/     过去：阶段快照与 Review
```

按需下钻：

- 长期原则与教学方法：[`core/`](core/README.md)
- Lemon 执行模板：[`templates/`](templates/README.md)
- 过去阶段与总结：[`history/`](history/README.md)

> **临时信息可以很多，正式结构必须很少。**
