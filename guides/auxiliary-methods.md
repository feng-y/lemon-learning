# Auxiliary Learning Methods

> Audience：家长、家庭指导老师、AI  
> Purpose：记录可以辅助主流程的方法，以及它们真正解决的问题和使用边界。  
> Principle：辅助方法不能反过来成为新的主流程中心。

## 1. A/B 交替听写

### 解决什么问题

避免只看“当天会不会”，增加延迟提取和旧内容随机回访。

### 怎么用

- A/B 两本按单双日交替；
- 两本不承担“新内容本/旧内容本”之类固定分工；
- 当天内容混合少量旧内容；
- 反复错误动态回流；
- 稳定内容逐渐退出高频复测。

### 边界

A/B 本是验证载体，不是罚抄本。

## 2. Retrieval Practice / 主动提取

### 解决什么问题

减少“看着会、离开材料就不会”的假掌握。

### 怎么用

先回忆，再看答案。

例如：

- 合上教材回忆昨天的 3–5 个词；
- 不看课文复述主要内容；
- 不看例题说出解题关系；
- 听英文后回忆对应意思。

### 边界

主动提取不是无限加测验。每次只需要少量、高频、短时确认。

## 3. Spaced Repetition / 延迟复习

### 解决什么问题

判断知识是否真正进入较稳定记忆，而不是只停留在短时记忆。

### 当前适合 Lemon 的轻量节奏

- 当天：即时确认；
- 第二天：盲测/主动回忆；
- 3–7 天：随机抽查；
- 稳定后：降低出现频率。

### 边界

不需要构建复杂的成人级间隔算法。重点是“错误会回来，掌握后会退出”。

## 4. 轻量费曼

### 解决什么问题

用输出确认孩子是否真的理解，而不是只会重复。

### 怎么用

把它简化成：

> “你来教我一下。”

语文：

- 这个字为什么在两个词里都能这样用？

数学：

- 为什么这里用这个运算？

英语：

- 这个词怎么读、什么意思、能放在哪句话里？

### 边界

不要求长篇讲课。10 岁孩子能用一两句话说清核心即可。

## 5. 深层转译

### 解决什么问题

避免孩子只接收提示，没有真正把信息转成自己的理解。

### 推荐方式

先让 Lemon 外显第一反应：

- “抛开对错，你第一眼怎么想？”
- “你觉得最容易错在哪里？”
- “你准备怎么跟同学解释？”

### 边界

如果孩子缺少新知识，应该直接教学。深层转译不是“永远不能告诉答案”。

推荐顺序：

> 尝试 → 定位缺口 → 必要时直接教 → Lemon 重新解释 → 相似任务验证。

## 6. Interleaving / 交错练习

### 解决什么问题

避免孩子只会在同一题型连续出现时机械套模板。

### 怎么用

当一个基础技能已经初步稳定后，少量混合：

- 不同运算混合；
- 新旧生字混合；
- 英语新词和旧词混合。

### 边界

基础动作尚未建立时，不要过早大量交错，否则只会增加认知负担。

## 7. 舒尔特方格

### 解决什么问题

可作为视觉扫描、持续注意和眼动搜索的辅助训练。

### 使用建议

- 时间短；
- 规则固定；
- 记录个人进步，不和别人比较；
- 每周适量即可。

### 边界

舒尔特方格不能替代真实学习任务中的：

- 数学读题；
- 阅读定位；
- 数字和符号检查；
- 学习持续性。

即使舒尔特成绩提高，也必须回到真实任务验证是否迁移。

## 8. 学习块 / Pomodoro

### 解决什么问题

帮助当前持续性不足的 Lemon 建立清晰的开始、持续和结束边界。

### 当前建议

- 约 20–25 分钟学习；
- 约 5 分钟休息；
- 开始前准备水、厕所、材料；
- 休息结束返回原任务。

### 边界

时间不是绝对规则。孩子高度投入时不需要为了计时器强行打断；明显疲劳时也可以调整。

重点是建立任务块，而不是迷信番茄钟。

## 9. 错题本

### 解决什么问题

保存真正有复习价值的错误模式。

### 当前建议

不是所有错题都抄进错题本。

优先记录：

- 同类错误反复出现；
- 暴露了概念缺口；
- 读题或关系错误有代表性；
- 心理退出导致的典型错误；
- 修复后值得再次验证。

### 边界

错题本的目标是减少未来同类错误，不是增加抄写工作量。

## 10. 方法选择原则

每引入一个方法，先问：

1. 当前真实问题是什么？
2. 这个方法具体解决哪一步？
3. 如何判断有效？
4. 什么时候可以停止使用？

如果回答不清楚，就不应该仅因为“这是一个流行学习方法”而加入 Lemon 的主流程。
