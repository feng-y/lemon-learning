# Templates

> Audience：Lemon 直接使用。  
> Purpose：把 Teaching Manual 中的方法压缩成短、固定、可重复的动作模板。  
> Not For：不解释教育理论，不出现 AI/Prompt/能力模型等成人术语。

模板只负责“今天怎么做”。真正书写发生在 A/B 听写本、任务手册、阅读本、作文本和练习册等执行载体中。

---

## 1. 错误字学习卡

仅用于听写错误、易混、反复忘记的字。

```text
今天修复的字：________

1. 我错在哪里？
□ 读音  □ 字形  □ 笔顺  □ 偏旁  □ 形近字  □ 意思

2. 课文里是什么意思？
________________________

3. 我查到另一个不同的意思：
________________________

4. 为什么两个意思都用这个字？
________________________

5. 我最容易写错的地方：
________________________

6. 我自己说一句：
________________________
```

## 2. 阅读定位卡

```text
今天阅读：____________

1. 这篇文章大概讲什么？
________________________

2. 题目问什么？
圈出关键词。

3. 答案可能在哪一段？
第____段

4. 圈出能证明答案的词或句子。

5. 我用一句完整的话回答：
________________________
```

核心口令：

> 不会答，先回原文找。

## 3. 三句话表达卡

```text
今天我想写：____________

第 1 句：发生了什么？
________________________

第 2 句：一个具体细节。
________________________

第 3 句：我觉得怎么样？为什么？
________________________
```

先说再写。能稳定完成三句话后，再逐步进入短段落。

## 7. 每日任务卡

实际建议优先使用已有任务手册，不额外增加新本子。

```text
今天要完成：
□ 数学
□ 语文
□ 英语
□ 阅读/表达

开始前：
□ 喝水
□ 上厕所
□ 材料准备好
□ 学习时不听故事

完成一项后：
1. 自己检查
2. 打勾
3. 看下一项

今天我最满意：____________
```

## 使用原则

- 模板不是每次全部使用；
- 普通任务走最短流程；
- 反复错误和困难任务才调用专项模板；
- 目标是模板最终被孩子内化，而不是永远依赖模板。
