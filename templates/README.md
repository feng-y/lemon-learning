# Execution Templates

> Audience：Lemon 直接使用，家长 / 老师负责选择何时调用。  
> Purpose：把 Teaching Manual 中的方法压缩成短、固定、可重复的执行动作。  
> Not For：不解释教育理论，不承担长期进度记录，不替代 A/B 听写本、任务手册、阅读本、作文本或错题本。

## 使用原则

模板不是每天全部使用，也不是新的大册子。

默认规则：

- 普通任务走最短流程；
- 反复错误、困难任务或尚未内化的动作才调用专项模板；
- 真正书写和学习记录留在线下执行载体中；
- 当 Lemon 已稳定形成相应动作后，模板应减少甚至退出。

当前模板：

1. [`daily-task-card.md`](daily-task-card.md)：每日任务启动与切换；
2. [`wrong-character-card.md`](wrong-character-card.md)：错误字 / 易混字深度修复；
3. [`reading-location-card.md`](reading-location-card.md)：语文阅读定位与证据；
4. [`three-sentence-expression-card.md`](three-sentence-expression-card.md)：从说一句到三句话表达；
5. [`math-read-card.md`](math-read-card.md)：数学应用题读题与关系提取；
6. [`math-check-card.md`](math-check-card.md)：数学提交前检查；
7. [`english-word-card.md`](english-word-card.md)：英语单词轻量输入—输出—延迟提取。
