# 英语单词轻量卡

> 用途：让一个核心词同时建立声音、意义、认读、表达和延迟回忆连接。

```text
Word：____________

1. 我会听：□
2. 我会读：□
3. 我知道核心意思：____________

4. 我能放进一句熟悉的话：
________________________

5. 我换一个词再说：
________________________

6. 我遮住答案还能回忆：□

7. 明天再检查：
□ 听到能认出
□ 看到能读
□ 不看答案能说 / 写
```

## 使用边界

- 不为每个词做长篇费曼解释；
- 优先用于核心词、反复忘记词和“会看不会听 / 会背不会用”的词；
- 不扩展生僻词义；
- 当词汇进入稳定使用后退出高频复习。
