# 错误字学习卡

> 只用于听写错误、易混、反复忘记或笔顺反复错误的字。

```text
今天修复的字：________

1. 我错在哪里？
□ 读音  □ 字形  □ 笔顺  □ 偏旁  □ 形近字  □ 意思

2. 正确写法和笔顺我确认了吗？
□ 是

3. 课文里是什么意思？
________________________

4. 我查到另一个不同的常见意思：
________________________

5. 为什么两个意思都用这个字？
________________________

6. 我最容易写错的地方：
________________________

7. 我自己说 / 写一个例子：
________________________

今天重新写：□
下一次 A/B 听写再检查：□
```

## 使用边界

- 普通生字不使用此卡，走轻量闭环即可；
- 默认只扩展“课文义 + 一个明显不同的常见引申义”；
- 不以大量抄写代替理解和延迟提取；
- 连续两次延迟正确后退出专项修复。
