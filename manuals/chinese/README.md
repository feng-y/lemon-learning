# Chinese Teaching Manual

> Audience：家长、家庭指导老师、AI  
> Purpose：指导如何训练 Lemon 的语文基础能力。  
> Not For：不是给 Lemon 直接阅读的教材；不复制人教版教材内容；不把所有生字、课文提前展开。

## 1. 核心定位

语文学习的主线不是“多记知识点”，而是让语言逐步生长：

> **字 → 词 → 句 → 篇 → 阅读 → 表达 → 写作**

当前三升四阶段只训练下一步能力，不提前进入复杂文学赏析。

## 2. 普通生字：轻量闭环

大多数生字只需要：

1. 在课文中认读；
2. 理解课文里的基本意思；
3. 确认正确笔顺；
4. 规范书写；
5. 组 1–2 个常用词；
6. 通过 A/B 听写交替验证。

达到“会读、会写、笔顺正确、知道课文义、延迟听写仍正确”即可退出，不对所有字做深度扩展。

## 3. 错误字：深度修复

只有以下情况进入专项修复：

- 听写写错；
- 笔顺反复错误；
- 形近字混淆；
- 会写但不理解；
- 同一个字连续出错；
- 离开短时记忆后马上忘记。

### 3.1 先判断错在哪里

错误类型可以是：读音、字形、笔顺、偏旁、形近字、词义或短时记忆依赖。

不要统一处理成“再抄十遍”。

### 3.2 字字生花：只做必要的语义扩展

目标不是一个字列十几个词，而是帮助孩子理解：**为什么同一个字能在不同语境中长出新的意思。**

每次默认只选两个明显不同的语义维度：

- 一个课文中的基本义；
- 一个最能体现语义迁移的引申义。

例如“潮”：

- 潮水：海水一波一波地涌动；
- 思潮：思想也像潮水一样一波一波地出现、传播。

关键问题不是“背两个词”，而是让 Lemon 说出：

> **为什么‘思潮’也有一个‘潮’字？**

这种语义桥梁比继续堆更多组词更有价值。

资料可由 Lemon 在《字词句手册》《教材全解》等材料中查找，也可由 AI 辅助生成扩义说明。AI 只负责按需扩展，不替代 Lemon 自己解释和使用。

### 3.3 错误字修复 SOP

1. 判断错误类型；
2. 重新确认字形、偏旁、结构和正确笔顺；
3. 理解课文义；
4. 查找一个不同维度的引申义；
5. 解释两个意思之间的共同特点；
6. 各说或写一个简短例子；
7. 当天重新写；
8. 下一次 A/B 听写时盲测；
9. 连续两次延迟正确后退出专项修复。

## 4. A/B 听写本

A/B 两本按单双日交替使用，不承担不同职责。

原则：

- 今天轮到哪一本就写哪一本；
- 内容由“当天新内容 + 少量旧内容随机抽查”组成；
- 错误内容动态重新出现；
- 已稳定掌握的内容退出高频复测。

A/B 本不是罚抄本，而是记忆验证载体。

## 5. 课文与古诗

当前阶段不追求复杂分析。

### 课文

训练孩子：

1. 读通；
2. 一句话说出主要讲什么；
3. 找出重要字词；
4. 能用自己的话复述主要内容；
5. 要求背诵的内容采用分段背诵 + 延迟抽查。

### 古诗

训练：

1. 正确朗读；
2. 理解最基本的画面和意思；
3. 分句背诵；
4. 第二天、几天后再次主动提取。

暂不要求大量背景知识和复杂鉴赏。

## 6. 阅读：初级目标是定位与证据

Lemon 当前阅读训练的核心不是“分析作者思想”，而是：

> **看懂题目问什么 → 回原文定位 → 找关键词/证据 → 组织简短答案。**

### 6.1 基础阅读 SOP

1. **整体感知**：一句话说文章大概讲什么；
2. **圈题目关键词**：题目到底问谁、什么、什么时候、为什么或怎么样；
3. **回原文定位**：先找到对应自然段；
4. **圈证据词句**：找到真正支持答案的内容；
5. **组织答案**：用一句完整的话回答，不脱离原文。

核心规则：

> **不会答题时，不先猜答案；先回到文章，找到答案在哪里。**

### 6.2 当前只训练三种核心感知

- 发生了什么；
- 事情的基本顺序；
- 人物或作者表达出的明显感受。

复杂中心思想、表达手法、深度写作意图留到后续阶段。

## 7. 表达与写作：渐进式输出

Lemon 当前写字意愿低，因此第一目标不是写出“好作文”，而是愿意、能够把想法写出来。

训练阶梯：

> **口头说一句 → 写一句 → 写三句 → 写一个短段落 → 再逐步进入完整习作。**

### 三句话支架

1. 发生了什么？
2. 我看见/听见/做了什么具体事情？
3. 我觉得怎么样？为什么？

每次只增加一个具体细节，不一次要求结构、修辞、好词好句全部到位。

### 初级修改只检查四件事

- 是否围绕题目；
- 事情是否说清楚；
- 顺序是否基本合理；
- 笔顺、错字和卷面是否达到提交标准。

## 8. 家长介入

家长的目标不是替孩子完成答案，而是判断在哪一步断掉。

阅读卡住时依次问：

1. 题目问什么？
2. 关键词是什么？
3. 你觉得答案可能在哪一段？
4. 哪一句能证明？

错误字卡住时问：

1. 你错的是哪一部分？
2. 这个偏旁和意思有什么关系？
3. 课文里这个字是什么意思？
4. 另一个词为什么也用这个字？

孩子没有相应基础知识时直接教学，不通过连续追问逼他猜答案。

## 9. 当前掌握标准

### 普通字

- 会读；
- 笔顺正确；
- 会写；
- 知道课文义；
- 延迟听写能正确。

### 阅读

- 能说一句大意；
- 能根据题目定位段落；
- 能找到关键词或证据句；
- 能组织一句不脱离原文的答案。

### 表达

- 能围绕题目说清楚；
- 能完成一句到三句话；
- 逐步能写一个短段落；
- 不因担心写不好而完全拒绝落笔。

## 10. AI 辅助 Prompt：错误字扩义

```text
请帮助小学四年级学生修复错误字“【字】”，所在课文是《【课文名】》。

只提供两个明显不同的语义维度：
1. 课文中的基本义；
2. 一个最能体现语义迁移的常见引申义。

重点解释：为什么这个字可以从第一个意思生长到第二个意思？
请使用孩子能理解的画面或类比，例如“思想像潮水一样一波一波地涌来，所以叫思潮”。

同时给出：
- 基础字形与偏旁提示；
- 一个易错书写点；
- 每个词义 1 个代表词；
- 一个让孩子自己解释的问题；
- 当天和下一次听写的 2 个主动提取问题。

不要罗列大量词语和生僻含义，控制在一页内。
```

## 11. 边界

本手册定义“怎么教”。

- 具体生字、词语和课文内容来自人教版教材、《字词句手册》《教材全解》等；
- A/B 听写本、阅读本、作文本属于执行载体；
- 给 Lemon 使用的简化步骤放在 `templates/`，不在本手册中混入大量儿童任务页。
