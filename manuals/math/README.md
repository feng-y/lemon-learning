# Math Teaching Manual

> Audience：家长、家庭指导老师、AI  
> Purpose：指导如何训练 Lemon 的数学基础能力，重点是信息处理、读题、关系建立、计算与检查。  
> Not For：不是题库；不按题型堆技巧；不把所有错误都归因于“马虎”。

## 1. 核心定位

数学学习的主线不是单纯计算，而是完整的信息处理闭环：

> **读题 → 定位 → 提取信息 → 建立关系 → 选择表示/模型 → 计算 → 检查**

Lemon 当前最突出的问题包括：

- 题目稍难时容易心理退出；
- 减法、除法出现明显不合理结果；
- 简单题也可能在整套难题中跟着崩；
- 应用题答非所问；
- 读题不完整；
- 不检查结果是否合理；
- 父母强压后短期表现改善，说明执行状态对结果影响很大。

因此训练重点是：**让思考过程可见，让错误能定位，让答案经过最基础的合理性检查。**

## 2. 计算：先建立数感，再追求速度

基础计算训练包含两类能力：

1. **硬算能力**：能按规范完成加减乘除；
2. **技巧能力**：能使用拆分、凑整、运算律等降低计算负担。

当前优先级：

- 先保证运算意义和基本规则稳定；
- 再训练常见口算技巧；
- 不因追求速度而跳过必要步骤。

### 2.1 计算最小闭环

1. 看清数字和符号；
2. 判断大致范围或大小关系；
3. 选择口算、竖式或脱式；
4. 规范书写；
5. 完成计算；
6. 用估算、逆运算或重新计算检查；
7. 判断结果是否合理。

例如：减法结果通常不应大于被减数。出现明显违背常识的结果时，任务尚未完成。

## 3. 读题：当前 ROI 最高的训练

读题不完整会让后续所有计算失效。

### 3.1 应用题读题 SOP

1. 先看最后问什么；
2. 圈出问题中的关键词；
3. 找出已知条件；
4. 标出单位；
5. 用自己的话复述“已知什么、要求什么”；
6. 再进入数量关系。

家长不要直接问“会不会”，而问：

- 这道题最后求什么？
- 哪几个条件和这个问题有关？
- 哪个数字对应什么量？

## 4. 信息提取

应用题先区分：

- 已知信息；
- 目标信息；
- 有用条件；
- 可能的干扰信息。

当前阶段不要求所有题都做复杂标记，但对于 Lemon 易错的题，必须让关键信息外显。

推荐最小留痕：

- 圈问题；
- 划条件；
- 必要时写单位或关键词。

## 5. 建立关系

数学真正开始于“条件之间是什么关系”。

家长引导顺序：

1. 先让 Lemon 说关系；
2. 能说清时再列式；
3. 说不清时使用图、线段图、表格或简单示意；
4. 缺概念时直接教学，不让孩子长期猜。

不要一看到应用题就直接问“用加法还是除法”。这种浅层提示容易让孩子等答案。

## 6. 模型与表示

不是所有题都要画图。

当文字信息较多、关系不清或 Lemon 连续出错时，再使用：

- 线段图；
- 简图；
- 表格；
- 数量关系式。

原则：**使用最少但足够的表示帮助理解。**

## 7. 思考留痕

Lemon 当前需要让父母和自己都能看到“到底在哪一步断了”。

不同题目使用不同程度留痕：

- 基础计算：估算/范围 + 计算 + 检查；
- 新概念：保留关键步骤；
- 应用题：问题 + 条件 + 关系 + 列式 + 单位/答句；
- 复杂题：增加图或表。

留痕的目标不是让卷面更复杂，而是防止“随机写一个答案也算做完”。

## 8. 检查

检查不是一句“再检查一下”，而是固定动作。

### 8.1 计算题

- 数字抄对了吗？
- 符号看对了吗？
- 数位对齐了吗？
- 结果大致合理吗？

### 8.2 应用题

- 最后的答案回答了题目吗？
- 单位对吗？
- 是否漏了一步？
- 把答案放回原题，意思通吗？

## 9. 错误分类与修复

不要统一写“粗心”。

建议至少分为：

1. **输入错误**：看错、抄错、漏条件；
2. **理解错误**：没读懂问题；
3. **关系错误**：数量关系判断错；
4. **概念错误**：缺少基础知识；
5. **计算错误**：运算或步骤出错；
6. **书写错误**：数位、布局、涂改影响结果；
7. **检查缺失**：明显离谱未发现；
8. **心理退出**：遇难后随意填写或整页崩溃。

修复时只针对真正断点。

## 10. 心理退出处理

当 Lemon 出现大量明显离谱答案时，先判断他是否还在真正思考。

可以问：

1. 这道题问什么？
2. 结果应该比哪个数大或小？
3. 你卡在哪一步？

若完全答不上来，先缩小任务：

> 先不算，只把题目读懂。  
> 只说最后求什么。  
> 再说第一个有用条件。

恢复进入任务后，再继续计算。

## 11. 训练量与难度

避免直接用整套偏难题测试抗压。

当前更适合：

> **2 道有把握题 + 1 道需要认真想的题**

逐步延长有效思考，而不是突然增加整套难题。

## 12. 家长介入

### 有基础但没启动思路

问：

- 题目最后求什么？
- 哪个条件最关键？
- 你为什么想到这个运算？

### 缺知识

直接讲解核心概念，讲完后换一道同结构题确认。

### 随意填写

明确：不会可以说明，随机答案不算完成。

### 情绪退出

可以短暂停，但任务不会取消；恢复后只继续当前最小步骤。

## 13. 当前掌握标准

### 计算

- 基础四则运算稳定；
- 能识别明显不合理结果；
- 必要步骤书写清楚；
- 有最基础检查动作。

### 应用题

- 能说出问题；
- 能找主要条件；
- 能解释为什么这样列式；
- 答案与题意对应；
- 单位和答句基本完整。

### 学习行为

- 稍难题不立即乱填；
- 能说出具体卡点；
- 错误后愿意返回断点修复。

## 14. AI 辅助 Prompt：错题归因

```text
你是一名小学四年级数学教学助手。请分析下面一道错题和孩子的作答过程。

目标不是直接给答案，而是判断错误发生在哪一环：
1. 输入/读题；
2. 信息提取；
3. 数量关系；
4. 概念；
5. 计算；
6. 书写；
7. 检查；
8. 心理退出/随意填写。

请输出：
- 最可能的 1–2 个错误类型；
- 判断依据；
- 家长只需要问的 2–3 个问题；
- 一个最小修复练习；
- 一道同结构但换数字的验证题。

不要长篇讲课，不要一次扩展多个知识点。
```

## 15. 边界

- 本手册定义“怎么训练数学能力”；
- 北师大版四上教材提供具体学习内容；
- 练习册和错题记录属于执行载体；
- 舒尔特方格可作为视觉扫描的辅助训练，但不能替代真实数学任务中的读题、关系和检查训练。
