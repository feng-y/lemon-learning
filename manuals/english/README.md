# English Teaching Manual

> Audience：家长、家庭指导老师、AI  
> Purpose：指导如何围绕人教版 PEP 四上，建立适合 Lemon 当前阶段的英语语言输入与输出闭环。  
> Not For：不以语法体系为主；不把英语简化成机械背单词；不追求一次掌握大量内容。

## 1. 核心定位

四年级英语的核心不是提前学习复杂语法，而是建立稳定的第二语言学习循环：

> **听 → 理解 → 跟读 → 认读 → 说 → 写 → 复习**

Lemon 当前需要的是：

- 每天有稳定、可理解的英语声音输入；
- 能把听到的声音和词义、单词拼写建立连接；
- 能用课本核心句型进行简单替换和口头表达；
- 通过 A/B 听写和延迟回忆验证是否真正保留；
- 不把一次性默写正确当成已经掌握。

## 2. 单词学习

单词不只做“中文 → 英文拼写”。

一个核心词至少建立四个连接：

1. 听到声音能识别；
2. 看到单词能读；
3. 知道基本意思；
4. 能在简单句型中使用。

### 2.1 当前推荐的最小闭环

1. 听老师/音频读；
2. 看图或语境理解；
3. 跟读；
4. 看词认读；
5. 遮住后主动回忆；
6. 写一次或进入 A/B 听写；
7. 第二天再次听辨或回忆。

每天新词数量宁少勿多，建议保持在可稳定完成的范围，例如 5–8 个核心词。

## 3. 句型学习

四上阶段重点是“会用句型”，不是系统讲语法术语。

训练顺序：

1. 听懂课本原句；
2. 跟读；
3. 理解句子在什么情境下使用；
4. 替换人物、地点、颜色、物品等关键词；
5. 自己口头说出一个新句子；
6. 必要时写一个简短句子。

家长不要一次讲太多语法规则。先保证孩子“听得懂、说得出、会替换”。

## 4. 听力

听力不是考试环节，而是英语学习的入口。

每天建议保留约 10 分钟可理解听力，同一段材料可以重复，不追求每天换新。

### 4.1 听力层级

1. 听音能指出对应图片/物品；
2. 听音能识别单词；
3. 听一句能抓住关键词；
4. 听懂熟悉句型的基本意思。

当前不追求长篇听力理解。

## 5. 跟读与发音

跟读用于建立声音模式。

建议：

- 先听完整句；
- 再模仿节奏和重音；
- 每次只练少量句子；
- 不因为一个音不完美反复纠缠太久。

目标是“敢开口、基本可辨认、越来越接近”，不是一开始追求播音级发音。

## 6. 阅读

当前英语阅读目标很初级：

- 看懂熟悉词和句型；
- 能把句子和图片/情境对应；
- 能回答非常直接的问题。

不要把中文阅读理解的复杂技巧直接搬到英语。

## 7. 口语输出

每天至少有极短输出。

例如：

- 回答一个课本问题；
- 用句型替换一个词；
- 看图说一句；
- 用 1–2 句介绍自己或物品。

输出很短也可以，但必须真实发生。

## 8. 书写

英语书写当前只要求：

- 字母形状清楚；
- 大小写基本正确；
- 单词间距清楚；
- 能够辨认。

不要因为书写不漂亮而无限返工。书写规范服务于可识别和正确拼写。

## 9. A/B 听写

A/B 两本按单双日交替使用，和语文一致。

英语听写可混合两种形式：

1. 听英文写英文；
2. 看中文/图片回忆英文。

同一词如果：

- 听得出但写不出：加强声音与拼写连接；
- 看得懂但听不出：加强听音识别；
- 单词会但句子不会：回到核心句型做替换。

当天写对不等于掌握。下一次 A/B 听写或第二天主动回忆仍正确，才更接近稳定掌握。

## 10. 复习与主动提取

复习优先采用“先回忆，再看答案”。

例如：

- 看图片说单词；
- 听音说意思；
- 给中文回忆英文；
- 给句型换一个词重新说；
- 第二天不看教材回忆昨天学过的 3–5 个词。

不要把复习变成一直重新看教材。

## 11. 费曼式记忆：只做轻量输出

费曼方法在四上英语中可以简化为：

> **“你来教爸爸这个词/句子。”**

例如一个单词：

1. 怎么读？
2. 是什么意思？
3. 在哪句话里能用？

一个句型：

1. 什么时候用？
2. 换一个词还能怎么说？

不要求 Lemon 做长篇解释。

## 12. 家长介入

### 孩子不会读

先播放正确声音，再跟读，不让他反复猜发音。

### 孩子会看不会听

增加听音指词、听音指图、听音口答。

### 孩子会背不会用

把单词放回课本句型，做最简单替换。

### 孩子拒绝开口

先允许极短回答，例如一个词或半句，再逐渐扩展。

## 13. 当前掌握标准

### 单词

- 听到能大致识别；
- 看到能读；
- 知道基本意思；
- 核心词能正确拼写；
- 能放进一个熟悉句型。

### 句型

- 能听懂熟悉句型；
- 能跟读；
- 能替换一个关键词；
- 能完成极短口头输出。

### 学习行为

- 每天有稳定听力输入；
- A/B 听写按节奏执行；
- 第二天有少量主动回忆；
- 不依赖一次性大量背诵。

## 14. AI 辅助 Prompt：PEP 单词轻量费曼卡

```text
请为人教版 PEP 四年级上册单词“【word】”生成一张适合 10 岁孩子的轻量费曼记忆卡。

只包含：
1. 单词和中文核心意思；
2. 一个最简单、最贴近 PEP 课本语境的例句；
3. 一个孩子可以回答的问题；
4. 一个“换词再说”的小练习；
5. 第二天的两个主动回忆问题。

要求：
- 不讲复杂语法术语；
- 不扩展生僻词义；
- 控制在一页内；
- 目标是让孩子能听、能读、能说、能写，而不是只记中文释义。
```

## 15. 边界

- 本手册定义“怎么训练英语”；
- 人教版 PEP 四上提供具体词汇、句型和课文内容；
- A/B 听写本属于执行载体；
- 后续如引入自然拼读、分级阅读等方法，应先确认是否解决当前真实问题，再单独扩展，不直接塞进主流程。
