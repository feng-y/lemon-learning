# Active Problems

> 这里不是当前学习主线。  
> 这里只跟进那些在**正常教材教学和针对训练之后，仍跨时间、跨任务持续阻碍学习**的问题。

默认顺序是：

```text
教材目标
→ 正常教学
→ 针对训练
→ 观察掌握
→ 仍持续阻碍
→ 才进入 Problems 跟进
```

一次错题、一次冲突、一次状态不好都不进入这里；临时观察先进入 `active/notes/`。

长期问题不只回答“有没有”，还要回答：

> **在哪些场景出现？当前需要多少支架？是否影响正常教学？支架是否正在下降？**

统一支架等级：S0 成人带着完成；S1 关键提问后继续；S2 依靠模板完成；S3 基本独立、少量提醒；S4 独立完成、自检并可迁移。

## LP-01 自主执行

**现象**：频繁询问下一步、学习中断多、需要父母持续调度。

**影响**：正常教材任务无法顺畅连续推进。

**当前目标**：自己看任务 → 启动 → 完成 → 检查 → 勾选 → 下一项。

**观察**：典型任务的支架等级、父母实时调度是否下降。

**退出条件**：大部分日常任务达到 S3–S4，父母退到抽查。

---

## LP-02 困难任务中的停留能力

**现象**：稍难任务后可能乱填、出现明显离谱答案，后续简单题质量也可能下降。

**工作假设**：难度上升后退出认真处理状态。该假设只用于指导干预，不作为人格判断。

**影响**：新知识、变式和需要思考的任务无法形成有效训练。

**当前目标**：不会可以明确说，但要停在具体卡点，尝试下一小步。

**观察**：是否能说出卡点；恢复任务需要什么支架；难题后后续任务质量是否保持。

**退出条件**：面对适度困难可在 S3–S4 下继续停留并推进。

---

## LP-03 数学问题解决中的持续断点

**现象**：读题不完整、漏条件、关系错误、答非所问、结果明显不合理仍未发现。

先确认它不是正常的新知识缺口：

1. **知识 / 模型没学会** → 回到 `curriculum/math.md` 正常教学；
2. **已经学会但持续不会调用** → 才在这里持续跟进。

**当前目标**：问题 → 条件 → 关系 → 表示 / 模型 → 计算 → 检查。

**观察**：典型题的支架等级；非知识性错误；相近新题是否能迁移。

**退出条件**：常规任务达到 S3–S4，专项读题 / 检查模板逐步退出。

---

## LP-04 提交质量

**现象**：笔顺、可识别书写、卷面、答题布局、漏题和提交前检查不稳定。

**影响**：已经掌握的知识不能稳定转化为合格输出。

**当前目标**：把基本书写、完整作答和提交前检查内化为“任务完成”的一部分。

**观察**：不同场景的支架等级；退回重做是否减少；是否主动自检。

**退出条件**：多数日常任务达到 S4，基本提交质量成为自动动作。

---

## LP-05 语言信息提取与表达的持续断点

**现象**：阅读答题可能脱离原文，不会定位证据；表达启动困难。

先通过语文 / 英语正常教学建立词句、阅读和表达能力；只有在相应知识已经教学后仍持续出现，才作为专项问题跟进。

**当前目标**：

> 阅读：题目问什么 → 原文定位 → 找证据 → 完整回答  
> 表达：先口头说清楚 → 组织 1–3 句 → 再落笔

**退出条件**：基础阅读定位和短表达达到 S3–S4，专项模板不再成为必需。

---

## LP-06 延迟保留与真实掌握

**现象**：部分语文 / 英语内容“当时会，过后不会”。

**影响**：教材推进形成表面完成，但知识没有真正积累。

**当前目标**：隔天、隔几天仍能主动提取，并在相近场景使用。

**观察**：当天、次日、隔日提取；提示程度；换场景能否使用。

**退出条件**：延迟提取和相近场景迁移稳定，复习频率自然下降。

---

## 维护规则

- Problems 是教学后的持续问题跟进，不是学习起点；
- 新知识不会首先回到 Curriculum，不进入 Problems；
- 一次错误首先进入正常反馈或 `notes/`；
- 只有反复、持续、明确影响正常学习时才升级；
- 每次正式 Review 只做：继续、调整、退出；
- 长期目标是随着正常教学质量提升，专项 Problems 越来越少。
