正常教学中只伴随三类能力训练：

1. **正确性 / 规范**：把基本动作做对并逐步自动化；
2. **学习耐力**：能更长时间专注、书写、背诵和思考；
3. **学习技巧**：学会更有效地处理学科任务。

它们不单独扩成新的课程体系，而是嵌入真实教材学习。持续跨周阻碍正常教学的问题，才进入 `problems.md`。

## 本月 Top 5

### 1. 任务组织：先减少“先做什么”的消耗
- 目标：面对多个任务时，不再长期卡在选择和纠结；先由成人帮忙梳理顺序，再逐步让 Lemon 自己参与排序。
- 做法：学习开始前只花 1–2 分钟，把任务写成清晰的 1→2→3；一次只看当前一项，完成再进入下一项。
- 介入：任务乱先理顺，不用连续催促替代任务组织；成人逐步从“替他排”退到“让他自己排后确认”。
- 检查：面对 2–3 个任务时，是否更快开始；是否减少反复问“下一步做什么”；成人催促次数是否下降。

### 2. 语文：字词、背诵积累、阅读三条基础线跑起来
- 目标：建立四上必背必默清单；普通生字轻量过关；阅读形成“题目→定位→证据→回答”。
- 技巧重点：关键错字 / 易混字使用“字字生花”建立字形—字义—常用词—语境的语义网络；背诵尝试关键词、画面、顺序和主动提取。
- 规范：汉字笔顺、字形和基本答题表达当场纠正，不靠大量罚抄。
- 检查：隔日能否提取；新短文能否按同样步骤找证据。

### 3. 数学：计算 / 数感打底，四上做认知预热
- 目标：确认三年级计算底座；训练拆分、凑整、估算和结果合理性；预热大数、线与角、乘法、运算律、除法。
- 技巧重点：数感、运算感、计算技巧、几何中的看 / 画 / 量 / 转化，以及应用题中的问题—条件—关系—模型—检查。
- 当前训练规则：**简单题慢做，复杂题真想，不会题重做。**
  - 简单题慢做：练读题完整、步骤规范、计算正确和结果检查，先稳定再提速；
  - 复杂题真想：不秒退，先说清问题、条件、关系和卡点；允许有限时间思考，必要时再给最小提示；
  - 不会题重做：看过提示或详解后必须脱离答案重新完成，必要时再做 1 道小变式或隔日复测。
- 规范：竖式、数位、单位、答句、几何作图和基本解题步骤保持正确。
- 检查：简单题是否越来越稳；复杂题是否能在困难里停留并说出卡点；不会题是否从“看懂”变成真正独立完成。

### 4. 英语：稳定第二语言学习闭环
- 目标：PU2 已学内容能脱离课堂输出；PU3 稳定推进；PEP 四上轻量衔接。
- 技巧重点：声音 → 词 → Chunk → 句型 → 语言规则；词汇尽量连接发音、词形、意思、场景和句子；规则从例句中发现并再次使用。
- 当前判断：词汇认读基础并不差，重点观察注意发散后能否把认读落实为提取、书写和实际使用。
- 规范：字母、大小写、单词拼写、间距、标点和基本语序及时修正。
- 检查：课堂后 30 分钟短确认；隔日能否提取并换相近语境使用。

### 5. 学习耐力：从短时间稳定专注开始
- 目标：先稳定完成 10–15 分钟单一任务，再逐步延长到 20–25 分钟。
- 训练对象：专注、连续书写、背诵、阅读和遇到困难后的思考停留。
- 介入：先确保任务顺序清楚，再要求持续专注；中断时恢复原任务，不用催促叠加压力。
- 检查：能否在不降低完成质量的前提下，逐步延长持续时间并减少提醒。

## 本周 Top 3

> 每周只选 3 件主任务。学科任务之外，本周把“任务组织”作为贯穿动作，不额外增加任务量。

### 任务组织｜开始前先排清楚 1→2→3
- 做：每天正式学习前，把当天任务压缩成最多 3 项并排好顺序；Lemon 只看当前一项。
- 教：前几次成人直接示范排序；随后让 Lemon 先排，成人只确认。
- 查：从看到任务到开始第一项需要多久；是否减少“我先做什么”的纠结和催促。
- 结果：`成人排 / 一起排 / Lemon 能自己排`

### 数学｜计算与运算感
- 做：每天 10–15 分钟，5–8 题；每组只针对一个目标，如凑整、拆分、估算或乘除基础。
- 原则：**简单题慢做，复杂题真想，不会题重做。**
- 教：先试 → 成人示范 1 个 → 一起做 1 个 → Lemon 独立做 → 最后 1–2 个小变式。
- 查：简单题是否稳定规范；复杂题是否能说出卡点而不是直接退出；讲过 / 看过的不会题能否独立重做。
- 结果：`未掌握 / 基本掌握 / 稳定`

### 英语｜PU3 跟进 + 课后短确认
- 做：正常上课；课后 30 分钟内确认当天核心词句。
- 教：听懂 / 跟读 → Chunk / 句型使用 → 必要时总结一个简单规则 → 换词表达。
- 查：第二天抽 3–5 个核心词句，看能否听懂、说出、读懂并在适用场景使用。
- 结果：`未掌握 / 基本掌握 / 稳定`

## 每次 Short Lesson：15–25 分钟

一次只教会一小步：

1. **先理任务**：当前只做哪一项，结束后下一项是什么；
2. **小目标**：结束时 Lemon 能做到什么；
3. **先试**：判断是不会、提醒后会，还是已经基本会；
4. **示范**：成人只示范一个核心动作或思路；
5. **练习**：模仿一个 → 自己一个 → 小变式一个；
6. **检查**：换一个相近新任务，不问“懂了吗”；
7. **反馈**：明确指出“原来哪里不会 / 不稳，现在具体哪里变会了”。

成人介入顺序：**任务乱先理顺 → 等一下 → 一个提示问题 → 示范一步 → 必要时重新教**。能自己走就不接管。

每次学习最多顺带抓：

- **1 个规范动作**，例如笔顺、竖式、英语大小写；
- **1 个耐力目标**，例如连续专注 15 分钟；
- **1 个技巧重点**，而且必须服务于当天主任务。

不要求三类每次都同时出现。
