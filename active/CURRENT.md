## 1. 当前阶段

- 时间：2026 年暑假
- 学段：三年级结束，即将进入四年级
- 年龄：10 岁
- 当前策略：**三年级复习确认基础 + 四年级适量预习暴露问题**

这不是“抢跑四年级”。复习和预习都只是诊断与训练载体，真正目标是修复会在四年级继续放大的学习断点。

## 2. 当前最重要的长期问题

### A. 自主执行

当前表现：

- 经常问“下一步做什么”；
- 学习过程容易中断；
- 需要父母持续调度。

当前目标：自己看任务、启动、完成、检查、勾选并进入下一项。

### B. 困难任务中的停留能力

当前表现：

- 题目稍难时可能心理退出；
- 出现乱填、明显离谱答案；
- 一道难题可能影响后续简单题质量。

当前目标：不会可以明确说，但要停在具体卡点，不退出整个任务。

### C. 数学信息处理

当前问题：读题不完整、关系不清、答非所问、结果明显不合理仍未发现。

当前训练闭环：

> 问题 → 条件 → 关系 → 表示 / 列式 → 计算 → 检查

### D. 提交质量

当前统一标准：

- 汉字笔顺正确；
- 内容清晰可识别；
- 不明显漏题；
- 数学布局和数位不影响正确性；
- 提交前有基本自检。

笔顺、卷面、布局、漏题不拆成多个碎目标，统一视为“任务完成质量”。

### E. 语言信息提取与表达

语文当前重点：

> 题目问什么 → 回原文定位 → 找证据 → 完整回答

表达当前阶梯：

> 口头一句 → 写一句 → 三句话 → 短段落

### F. 延迟保留

当前问题：部分语文 / 英语内容可能“当时会，过后不会”。

当前目标：从短时记忆转为隔一段时间仍能主动提取和使用。

## 3. 当前三科学习重点

### 语文

- 三年级常错字、易混字、延迟听写；
- 四上生字轻量预习，不对所有字深挖；
- 错误字才进入深度修复；
- 阅读训练定位与证据；
- 表达从一句到三句逐步推进。

### 数学

- 三年级四则运算、口算、数感与结果合理性；
- 应用题完整读题；
- 四上只选少量新内容作为诊断样本；
- 先保证信息处理闭环，再增加难度和题量。

### 英语

- 当前从 PU2 向 PU3 过渡，同时衔接 PEP 四上；
- 修复“课堂看似学过、课后无法独立输出”；
- 建立听 → 理解 → 跟读 → 认读 → 说 → 写 → 复习闭环；
- 网课或明确教学后 30 分钟内做短确认；
- A/B 交替用于延迟提取验证。

## 4. 当前父母介入原则

先判断是哪种状态：

- 不会：直接教学；
- 有基础但没启动：只问一个能启动思路的问题；
- 注意中断：恢复环境和任务；
- 心理退出：缩小到当前最小步骤；
- 明显敷衍：退回对应部分，不用情绪加码。

长期方向：

> 父母示范 → 提醒 → Lemon 自检 → 父母抽查 → Lemon 独立

## 5. 这一阶段怎么看进步

优先观察：

- “下一步做什么”是否减少；
- 父母提醒是否下降；
- 学习块中断是否减少；
- 遇难后乱填是否减少；
- 是否更能说出具体卡点；
- 数学非知识性错误是否下降；
- 自检发现错误是否增加；
- 阅读定位和证据意识是否稳定；
- A/B 听写延迟保留是否提高；
- 不提高父母情绪强度，任务是否仍能完成。

## 6. 当前阶段结束 Gate

暑假结束时重点判断：

1. 大部分日常任务是否能自己启动；
2. 遇到困难是否不再明显乱填；
3. 提交前是否形成基本自检；
4. 语文是否形成基础阅读定位；
5. 数学是否能完整经历读题到检查；
6. 英语是否形成输入、输出和延迟复习的最小闭环。

未稳定通过的部分进入四年级继续跟踪；已稳定通过的部分降低父母介入。
