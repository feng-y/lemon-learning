## 2. 当前教学主线

### 语文

围绕四个长期系统推进：

1. 字词：识字、正确笔顺、课文义、常用词和易错字；
2. 背诵与积累：古诗、日积月累、要求背诵内容和优秀表达；
3. 阅读理解：大意、题目理解、原文定位、证据和完整回答；
4. 表达与写作：先口头组织，再从一句、三句逐步走向段落和习作。

当前重点不是只修错字，而是建立完整的语文学习系统。

### 数学

以教材单元知识结构为主线：

> **概念理解 → 数感 → 计算 → 几何与空间 → 数量关系 / 模型 → 问题解决 → 检查与迁移**

当前重点：

- 三年级基础计算和常用口算方法；
- 数的大小、数量级、拆分组合和结果合理性；
- 基础几何与空间关系；
- 四上单元知识地图、核心概念和典型模型；
- 少量有目的训练，用来暴露薄弱点和盲点，而不是无目的刷题。

### 英语

两条学习线并行：

- PEP 四上：校内教材主线；
- Power Up 3：听说读和真实语言能力拓展。

长期结构：

> **语音 → 词汇 → Chunk / 固定搭配 → 句型 → 基础语言规则 → 听说读写 → 阅读理解与基础表达 → 长期积累**

基础语言规则包括人称、be 动词、单复数、介词、语序、疑问 / 否定，以及随教材逐步建立的基础时态意识。

## 3. 统一教学方式

三科都采用：

> **教材目标 → 引导理解 / 明确示范 → 少量针对训练 → 暴露盲点 → 针对修复 → 延迟验证 / 迁移 → 正反馈 → 减少支架**

原则：

- 缺知识就直接教；
- 有知识但不会调用，再做引导和支架；
- 训练必须有明确目的；
- 做对一次不等于掌握；
- 鼓励基于真实进步和具体行为，不做空泛表扬。

## 5. 当前如何判断进步

同时看两类结果：

### 教材掌握

- 该学的知识是否系统覆盖；
- 核心概念是否理解；
- 重点内容是否能延迟提取；
- 单元知识是否开始形成结构。

### 能力成长

- 是否需要更少外部支架；
- 遇到困难是否还能留在任务中；
- 完成质量和自检是否提高；
- 隔一段时间或换相近场景后是否仍会。
