## 未来 1 个月 Top 5

### 1. 语文：建立四上“必背必默 + 字词 + 阅读”基础地图

本月只关注：

- 把四上需要背诵 / 默写 / 日积月累的内容单独列清；
- 普通生字轻量预习，反复错字才深修；
- 阅读稳定训练“题目 → 定位 → 证据 → 回答”。

**完成标准**：形成一份可执行的四上积累清单；A/B 听写开始运行；阅读定位动作能在提示或模板下完成。

### 2. 数学：先补底座，再预热四上重点

本月只关注：

- 三年级四则运算、常用口算和结果合理性；
- 数感：大小、数量级、拆分组合、估算；
- 四上优先预热“认识更大的数 / 线与角 / 乘法 / 运算律 / 除法”中的前置能力与核心问题。

**完成标准**：明确 Lemon 在计算、数感、几何三条线的主要薄弱点；针对性训练后能看到 1–2 个稳定改善，不追求刷完整册。

### 3. 英语：稳定第二语言学习节奏

本月只关注：

- PU2 已学内容能否脱离课堂输出；
- PU3 稳定跟进；
- PEP 四上做轻量词汇、句型和语言规则衔接；
- 保持“听懂 → 说出 → 读懂 → 基础书写”的闭环。

**完成标准**：课堂后 30 分钟短确认稳定执行；核心词句能隔日提取；PEP 与 PU3 分工清楚。

### 4. 学习执行：减少父母实时调度

本月只关注：

- 开始前看任务；
- 一个学习块内减少无关中断；
- 完成后基本自检；
- 不会时说出具体卡点，而不是乱填。

**完成标准**：同类日常任务的父母提醒开始下降。

### 5. 正反馈：让训练产生“我真的变会了”的体验

本月只关注：

- 反馈具体进步，不做空泛表扬；
- 同类任务在针对训练后明显变顺时及时指出；
- 不把题量当成努力证明。

**完成标准**：至少能明确指出几次“原来不会 / 不稳 → 现在能独立完成”的真实进步。
