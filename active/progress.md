## 当前阶段观察维度

只有在影响教学判断时才记录：

- **独立性**：同类任务需要的父母提醒是否下降；
- **困难停留**：遇到难题能否说出卡点并继续；
- **完成质量**：是否形成基本书写、完整作答和自检；
- **保留与迁移**：隔日、减少提示或换相近场景后是否仍会；
- **学习耐力**：在不过度疲劳的情况下，单任务持续时间是否自然增长；
- **自我效能**：是否越来越能感受到“我通过方法和练习把不会变成会”。

不要求每周全部记录。

## 周 Review 规则

1. 没有新证据就不写；
2. “做了多少页 / 多少题”不是核心进度，除非它能说明掌握变化；
3. 同一问题连续跨周、跨任务影响正常教学，才进入 `problems.md`；
4. 已稳定掌握的内容不持续占据 Progress；
5. 周 Review 的目标是决定下一步，不是评价 Lemon 好或不好。

## 月度 Learner / System Review

每月额外做一次很轻的反向检查，同时评估 Lemon 和这套支持系统：

- **能力**：基础、方法和独立性是否真的在增长？
- **兴趣 / 胜任感**：是否有更多真实成功体验，愿意继续尝试？
- **挑战耐受**：遇到不会时是否更能停留，而不是更害怕犯错？
- **负荷**：是否出现持续疲劳、明显抵触或学习时长不断膨胀？
- **支架**：家长 / 家教提醒是在减少，还是系统让 Lemon 更依赖成人？
- **效率**：哪些训练产生了真实掌握，哪些只是打卡、题量或重复劳动？

月度结论只允许三类动作：

> **继续有效做法 / 减少无效负担 / 调整一个最高优先问题。**

阶段结束时只提炼关键变化进入 `history/`。
