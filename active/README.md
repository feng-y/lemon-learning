# Active

这里是当前阶段的**执行工作面**。

日常只需要先看 [`CURRENT.md`](CURRENT.md)。

```text
CURRENT.md     这月 / 这周做什么，怎么教，怎么查
curriculum/    需要时查教材地图和学科重点
progress.md    每周记录真实掌握变化
problems.md    只跟进持续阻碍正常教学的问题
notes/         临时想法、一次观察、未确认信息
```

## 最小运行方式

```text
周初：CURRENT 选 1–3 个重点
  ↓
平时：直接教学和训练
  ↓
遇到不会教：查 curriculum / core
  ↓
周末：10 分钟检查结果
  ↓
高信号变化写 progress
  ↓
连续跨周的问题才进 problems
  ↓
更新下周 CURRENT
```

## 三条约束

1. **当前执行重点最多 5 个，本周主任务最多 3 个。**
2. **Curriculum 可以完整，但不能全部同时执行。**
3. **一个问题没有连续影响正常教学，就不升级为专项治理。**

> 目标不是维护文档，而是让家长能随时知道：**现在做什么、什么时候介入、怎么判断有效、下一步怎么调。**
