## 三类伴随训练

它们都服务于正常教材学习，不单独扩成新的复杂体系。

### 1. 正确性 / 规范

把基本动作做对并逐步自动化。

- 语文：正确笔顺、字形、基本答题表达；
- 数学：竖式、数位、单位、答句、作图和解题步骤；
- 英语：字母、大小写、拼写、间距、标点和基本语序。

错误优先即时、少量修正，不靠无目的罚抄和重复。

### 2. 学习耐力

像体力一样渐进增长：先稳定 10–15 分钟，再逐步延长到 20–25 分钟。

包括专注耐力、连续书写、背诵、阅读，以及遇到困难后继续思考和说出卡点的能力。

### 3. 学习技巧

只教能让当前任务学得更好的具体方法。

- 语文：关键字用“字字生花”建立语义网络；背诵用关键词 / 画面 / 顺序 / 主动提取；阅读用定位—证据；
- 数学：数感、运算感、计算技巧、几何中的看 / 画 / 量 / 转化，以及问题—条件—关系—模型—检查；
- 英语：声音—词—Chunk—句型—语言规则，连接发音、词形、意思、场景和真实使用。

## 执行约束

1. **当前执行重点最多 5 个，本周主任务最多 3 个。**
2. **每次 Short Lesson 只教会一个小目标。**
3. **每次最多顺带抓 1 个规范动作、1 个耐力目标、1 个技巧重点，不要求全部出现。**
4. **Curriculum 可以完整，但不能全部同时执行。**
5. **Problems 是异常升级机制，不是日常学习主线。**
6. **如果 PROFILE 中的负荷风险信号持续出现，优先减负和调整介入方式，而不是继续加任务。**

> 目标不是维护文档，而是让家长或家教随时知道：**Lemon 现在是什么状态、今天教什么、怎么介入、怎么检查、下一步怎么调。**
