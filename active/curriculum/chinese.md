# Chinese Curriculum — 人教版四年级上册

> 当前阶段：三升四暑假衔接  
> 定位：以教材为主线，把语文拆成“字词基础、背诵积累、课文理解、阅读理解、表达写作”五条长期能力线。问题修复只是辅助，不再作为主轴。

## 1. 语文完整学习结构

```text
识字与写字
→ 词语与句子
→ 课文 / 篇章理解
→ 背诵与长期积累
→ 阅读理解
→ 口头表达
→ 习作写作
```

语文最终要形成五个长期系统。

### A. 字词系统

普通生字按轻量闭环：

> 认读 → 正确笔顺 / 字形 → 课文义 → 常用词 → 句中理解 → 延迟听写

重点关注：

- 课内生字和二类字的认读；
- 正确笔顺和规范书写；
- 重点词语在课文中的实际含义；
- 常用搭配、近义 / 易混词；
- 易错字、易混字、反复遗忘字。

只有反复错误或值得理解的字进入“字字生花”深度修复，不对所有字过度展开。

### B. 背诵与长期积累系统

需要从教材中单独抽出一张“必背必默 / 长期积累地图”，不把背诵散落在每天任务中。

目前根据现有四上材料，重点至少包括以下类型：

#### 课文 / 古文背诵

- 《观潮》重点背诵段落（当前材料标注第 3、4 自然段）；
- 《精卫填海》全文；
- 《王戎不取道旁李》全文；
- 教材要求背诵的其他重点课文 / 段落。

#### 古诗词

- 《暮江吟》；
- 《题西林壁》；
- 《雪梅》；
- 《凉州词》；
- 《出塞》；
- 教材“日积月累”及单元要求的其他古诗词。

#### 日积月累 / 语言材料

- 与声音有关的词语；
- 与学习有关的名言；
- 秋天气象谚语；
- 与人物、故事有关的词语；
- 三字惯用语；
- 八字成语；
- 神话故事相关知识；
- 长城、大运河等资料性知识。

背诵闭环：

> 理解 → 熟读 → 背诵 → 默写 / 复述 → 次日提取 → 3–7 天回访 → 后续阅读 / 表达中再次调用

背诵的目标不是“一次打勾”，而是逐渐形成可长期调用的语言材料库（Language Bank）。

### C. 课文与篇章理解系统

每篇课文至少回答：

1. 写了什么？
2. 内容按什么顺序展开？
3. 哪些词句最重要？
4. 这些词句为什么这样写？
5. 这一篇可以学到一个什么表达方式？

当前阶段不做成人化文学赏析，重点是让 Lemon 真正理解课文内容与表达。

### D. 阅读理解系统

基础闭环：

> 读懂大意 → 看清题目 → 回原文定位 → 找关键词 / 证据 → 组织完整回答

逐步增加：

- 事件与顺序；
- 原因与结果；
- 人物与行为；
- 明显情感；
- 段落关系；
- 简单概括。

### E. 表达与习作系统

长期成长线：

> 看 / 读 / 经历 → 口头说清楚 → 1 句话 → 3 句话 → 短段落 → 多段组织 → 完整习作

当前优先：

- 有内容可说；
- 不跑题；
- 顺序基本清楚；
- 至少有一个具体细节；
- 写完自己读一遍并修改一个最明显的问题。

阅读和背诵中的优秀表达应逐步进入自己的表达，但不要求生硬套用。

## 2. 四上教材应维护的三张核心地图

### 2.1 单元学习地图

每个单元直接列：

- 核心课文；
- 生字 / 重点词 / 易错字；
- 课文理解重点；
- 阅读能力目标；
- 语文园地中的词句段运用；
- 习作目标；
- 单元结束后需要延迟复测的内容。

### 2.2 必背必默地图

单独维护整册所有：

- 课文 / 重点段落；
- 古诗文；
- 日积月累；
- 名言、谚语、成语和重要语言材料。

### 2.3 长期语言积累池

跨单元、跨学期持续积累：

- 古诗；
- 名句；
- 成语；
- 优秀词句；
- 优秀段落；
- 可迁移到表达中的语言材料。

## 3. 暑假衔接方式

### 复习三年级

- 反复错误字词；
- 基础背诵是否隔时仍能提取；
- 阅读是否能根据问题回原文；
- 一句到三句话的基础表达。

### 预习四上

优先做：

1. 浏览整册目录和单元结构；
2. 建立必背必默清单；
3. 生字做轻量认读和正确书写预热；
4. 古诗 / 古文先理解背景和大意，再开始熟读；
5. 课文以“读懂”为目标，不提前做复杂文学分析；
6. 习作只做口头素材准备和观察积累，不提前写整册作文。

---

## 4. 掌握标准

一个语文内容真正进入“掌握”，至少满足其中适用部分：

- 当时能读懂 / 写对 / 背出；
- 减少提示后仍能完成；
- 次日或隔日仍能提取；
- 换一篇短文仍能使用阅读方法；
- 在新的表达场景中能调用部分已积累语言。

具体单元推进和掌握变化记录在 `../progress.md`。
