# Chinese Curriculum — 人教版四年级上册

> 当前阶段：三升四暑假衔接  
> 目标：以教材为主线建立完整语文学习系统，而不是只围绕错字和阅读问题做修补。

## 1. 语文能力主线

```text
识字与写字
→ 词语与句子
→ 课文理解
→ 背诵与长期积累
→ 阅读理解
→ 口头表达
→ 习作写作
```

语文学习长期维护四个系统：

### A. 字词系统

- 课内生字：认读、正确笔顺、规范书写、课文义；
- 常用词语：理解搭配和语境，不只机械组词；
- 易错字 / 易混字：反复出现才进入深度修复；
- 重点字：必要时用“字字生花”理解不同常见语义之间的联系。

普通字轻量学，反复错误字深度修，不对所有字过度展开。

### B. 背诵与积累系统

覆盖教材中值得长期保留的内容：

- 古诗词；
- 日积月累；
- 要求背诵的课文 / 段落；
- 优秀词句和表达。

闭环：

> **理解 → 熟读 → 背诵 → 默写 / 提取 → 次日或隔日复现 → 长期间隔回访**

背诵不是一次打勾，而是形成可长期调用的语言材料。

### C. 阅读理解系统

当前优先建立基础高价值能力：

> **读懂大意 → 看清题目 → 回原文定位 → 找关键词 / 证据 → 组织完整回答**

随着年级推进，再逐渐增加：段落关系、人物与事件、情感、表达效果和篇章结构。

### D. 表达与写作系统

先有内容，再组织，再落笔：

> **看 / 读 / 经历 → 口头说清楚 → 组织 1–3 句 → 写下来 → 自己读一遍 → 修改最明显问题**

长期阶梯：

> **口头一句 → 口头三句 → 写一句 → 写三句 → 短段落 → 完整习作**

阅读和背诵中的优秀表达应逐步迁移到自己的表达中，但不强求生硬套用。

## 2. 每个单元怎么学

每个教材单元至少关注：

1. 本单元主题和核心课文；
2. 本单元生字、重点词语和易错字；
3. 要求背诵 / 默写 / 日积月累的内容；
4. 阅读训练重点；
5. 语文园地中的词句段运用；
6. 本单元习作目标；
7. 单元结束时哪些内容需要延迟复测。

## 3. 暑假衔接重点

复习三年级：

- 反复错误字词；
- 基础背诵是否隔时仍能提取；
- 阅读是否能根据问题回原文；
- 一句到三句话的基础表达。

预习四上：

- 建立课文和单元地图；
- 生字只做轻量预习；
- 提前识别必背、必默和长期积累内容；
- 不提前进行超出当前能力的复杂文学分析。

## 4. 掌握标准

一个内容真正进入“掌握”，至少满足其中适用的部分：

- 当时能完成；
- 减少提示后仍能完成；
- 次日或隔日仍能提取；
- 换一篇短文或换一个表达场景仍能使用。
