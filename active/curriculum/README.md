- [`english.md`](english.md)：人教 PEP 四年级上册 + Power Up 3。

## 使用原则

Curriculum 可以完整，但**当前执行必须很窄**。

- Curriculum 负责保存整册 / 整阶段的完整知识地图；
- `CURRENT.md` 每月只选择 Top 3–5 个重点；
- 日常任务只围绕当月重点推进；
- 新发现的薄弱点只有持续阻碍学习时才进入 `problems.md`；
- 不因为地图完整，就要求同时执行所有内容。

统一教学模型：

> **教材目标 → 引导理解 / 示范 → 少量针对训练 → 确认掌握或暴露盲点 → 针对修复 → 延迟验证 / 迁移 → 正反馈**

长期要求只有两条：

1. **地图完整**：知道这一阶段有哪些知识和能力，避免盲区；
2. **执行聚焦**：任何一个月只抓少数真正重要的目标。

> **完整性属于知识地图，优先级属于 CURRENT，执行不能被文档复杂度拖垮。**
