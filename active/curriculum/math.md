# Math Curriculum — 北师大版四年级上册

> 当前阶段：三升四暑假衔接  
> 目标：以教材单元知识结构为主线，以数学理解和数感为底层，以计算与几何为空间较大的核心训练面，通过诊断发现薄弱点和盲点，再做针对训练。

## 1. 数学能力主线

```text
概念理解
→ 数感
→ 计算
→ 几何与空间
→ 数量关系 / 典型模型
→ 问题解决
→ 检查与迁移
```

其中：

- **数感**：对数的大小、数量级、拆分组合、结果范围和数量关系有基本直觉；
- **计算**：既会规范硬算，也能识别凑整、拆分、运算律等更高效路径；
- **几何与空间**：通过看、画、量、拼、转化理解图形和空间关系；
- **模型与问题解决**：把题目中的条件转成关系和表示，再选择运算与方法。

## 2. 每个单元怎么学

每个单元不以“刷完多少题”为目标，而建立一张单元知识地图：

1. 单元在整册中的位置与权重；
2. 教材篇幅 / 课时占比作为重点程度的重要信号之一；
3. 核心概念；
4. 前置知识；
5. 必会技能；
6. 核心计算或几何能力；
7. 典型模型与代表性例题；
8. 常见盲点和易错点；
9. 与后续单元的连接；
10. 达到什么程度算过关。

教材页数可以帮助判断重点，但不能单独决定权重；还要看该知识是否是后续学习的基础。

## 3. 单元教学闭环

```text
回归课本
→ 理解概念和例题
→ 建立单元知识地图
→ 精做少量典型题
→ 暴露薄弱点 / 盲点
→ 判断真正断点
→ 针对训练
→ 变式验证
→ 单元总结与思维导图
```

训练的目的不是增加题量，而是验证理解、形成熟练度和暴露盲点。

少量训练可以刻意覆盖：

- 基础概念；
- 基本方法；
- 易错点；
- 变式；
- 综合迁移。

拒绝无诊断目的的重复刷题，但保留有目标的刻意训练。

## 4. 诊断与修复

遇到错误先判断：

- 概念没理解 → 回到概念和示范；
- 数感不足 → 做大小、范围、拆分和估算训练；
- 计算不熟 → 练基本计算或针对性口算技巧；
- 几何关系不清 → 画图、量、操作、转化；
- 模型不会 → 回到典型模型；
- 会模型但不会调用 → 做少量变式和关系提取；
- 读题不完整 → 训练问题、条件和单位；
- 困难后退出 → 缩小到最小下一步，恢复任务状态。

错误只是诊断信号，不自动等于“再刷十道同类题”。

## 5. 思维导图

每个重点单元结束后形成一张简洁思维导图，只回答：

> **这一单元学了什么？最重要的概念是什么？它们之间怎么连接？遇到什么问题用什么方法？**

思维导图服务于知识结构，不追求装饰和复杂排版。

## 6. 暑假衔接重点

复习三年级：

- 四则运算和常用口算；
- 数的大小和结果合理性；
- 基础几何与空间概念；
- 常见应用题中的数量关系。

预习四上：

- 先建立整册和重点单元地图；
- 选择有代表性的内容理解和练习；
- 观察新知识暴露的是旧基础缺口、数感不足，还是模型调用问题；
- 不以提前刷完整册为目标。

## 7. 掌握标准

真正掌握通常表现为：

- 能解释核心概念；
- 基础计算或几何操作稳定；
- 典型题能独立完成；
- 变式题仍能识别核心关系；
- 结果明显不合理时能产生警觉；
- 隔一段时间仍能完成相近任务。

具体单元推进和掌握变化记录在 `../progress.md`。
