# English Curriculum — PEP 四年级上册 + Power Up 3

> 当前阶段：三升四暑假衔接  
> 目标：PEP 负责校内教材主线与基础掌握，Power Up 3 负责更丰富的听说读与真实语言能力拓展。

## 1. 英语能力主线

```text
语音 / Phonics
→ 词汇
→ Chunk / 固定搭配
→ 句型
→ 基础语言规则
→ 听说读写
→ 阅读理解与基础表达
→ 长期积累
```

基础语言规则包括：

- 人称和代词；
- be 动词和常见动词变化；
- 单复数；
- 冠词；
- 常见介词；
- 基础语序；
- 疑问和否定；
- 随教材逐步建立的基础时态意识。

当前阶段不把英语学成脱离语境的语法体系。规则应从真实句子和教材场景中逐步抽象：

> **先听懂 / 读懂 → 观察相似句子 → 发现规律 → 成人做简单总结 → 替换表达 → 在新场景中再次使用**

## 2. 两条学习线

### PEP 四上：校内主线

重点关注：

- Unit Theme；
- Core Words；
- Useful Expressions / 核心句型；
- 听说活动；
- Phonics / Spell；
- Read and Write；
- 本单元出现的基础语言规则；
- Unit Review 和延迟复习。

目标是教材内容真正掌握，而不是只完成课堂和作业。

### Power Up 3：能力拓展

重点关注：

- Vocabulary；
- Listening；
- Speaking；
- Reading；
- Communication；
- 在更丰富语境中复现已学词句和基础规则。

Power Up 3 不替代 PEP 校内主线，也不要求和 PEP 单元机械对齐。

## 3. 长期学习系统

### A. 词汇与 Chunk

核心词至少建立：

- 听到能识别；
- 看到能读；
- 理解核心意思；
- 能放进熟悉句型；
- 需要时能正确拼写。

优先学习高频词组和固定搭配，而不是只记孤立中文释义。

### B. 句型与语言规则

句型先在语境中会听、会说、会替换，再逐步理解规则。

例如时态、介词等不是单独背规则结束，而要经历：

> 语境理解 → 规则发现 → 简单归纳 → 多次使用 → 阅读中再识别 → 写作中尝试输出。

### C. 听说读写

当前优先级：

> **听得懂 → 说得出 → 读得懂 → 能写基本内容**

保持短时、可理解、可重复的输入，并保证真实输出发生。

### D. 阅读理解与基础写作

阅读先训练：

- 看懂主题和基本信息；
- 从文本找到答案依据；
- 理解熟悉词句在新语境中的意思。

写作从句型替换、1–3 句真实表达逐步推进，不急于成人化作文。

### E. 长期积累

值得长期保留的内容包括：

- PEP 核心词句；
- Power Up 3 中高频、可复用表达；
- 简短对话和适合背诵的短文；
- 反复出现的基础语言规则。

闭环：

> **理解 → 跟读 / 使用 → 主动提取 → 次日复现 → 换场景使用 → 间隔回访**

## 4. 暑假衔接重点

- PU2 已学内容是否能脱离课堂独立输出；
- PU2 → PU3 平稳过渡；
- PEP 四上核心词句做轻量衔接；
- 网课或明确教学后 30 分钟内做短确认；
- A/B 交替用于语文和英语的延迟提取，不承担“新旧内容分本”的固定职责。

## 5. 掌握标准

一个核心词句或规则真正掌握，至少逐步满足：

- 能听懂；
- 能认读；
- 能理解；
- 能在熟悉句型中使用；
- 减少提示后仍能输出；
- 隔天或隔几天仍能提取；
- 换一个相近语境仍能使用。
