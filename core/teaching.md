# Teaching Methods

> Audience：家长、家庭指导老师、AI  
> Purpose：维护可跨阶段复用的教学方法。  
> Not For：不描述 Lemon 当前状态，不记录具体教材进度，也不保存临时想法。

## 1. 跨学科统一教学模型

所有学科先有正常教学主线，再通过真实学习发现薄弱点：

> **教材知识结构 → 明确阶段目标 → 引导理解 / 明确示范 → 少量针对训练 → 暴露薄弱点与盲点 → 针对修复 → 延迟验证与迁移 → 正反馈 → 减少支架**

关键原则：

- **教材决定覆盖范围**：先知道这一阶段应该学什么；
- **能力体系决定学习深度**：不是“做过”就等于掌握；
- **缺知识直接教**：新概念、新方法、新词义、新句型，不让孩子靠连续猜测获得；
- **训练必须有目的**：拒绝无诊断目的的重复刷题，但保留有明确目标的刻意训练；
- **模板只是临时支架**：目标是最终不看卡也能完成；
- **一次做对不等于掌握**：减少提示后仍会、隔天仍会、换相近场景仍会，才更接近真正掌握；
- **问题是反馈，不是主线**：只有持续阻碍正常教学的问题才进入专项跟进。

支架观察：

| 等级 | 状态 |
| --- | --- |
| S0 | 成人示范或带着完成 |
| S1 | 关键提问后才能继续 |
| S2 | 依靠模板 / checklist 完成 |
| S3 | 基本独立，只需少量提醒或最后检查 |
| S4 | 独立完成、自检，并能在相近任务中迁移 |

支架等级只用于判断成人下一步应该“教更多”还是“退后一点”。

## 2. 语文：语言深化与长期积累

语文主线：

> **识字写字 → 词语 → 句子 → 课文 / 篇章 → 背诵积累 → 阅读理解 → 口头表达 → 写作**

### 字词系统

普通生字走轻量闭环：

> 课文中认读 → 理解课文义 → 正确笔顺 → 规范书写 → 1–2 个常用词 → 延迟确认

只有错误字、易混字、反复忘记字才进入深度修复。必要时用“字字生花”理解不同常见语义之间的联系，但不堆大量词义和罚抄。

### 背诵与长期积累

长期积累包括：古诗、日积月累、要求背诵的课文 / 段落、优秀词句和表达。

闭环：

> **理解 → 熟读 → 背诵 → 主动提取 / 默写 → 次日或隔日复现 → 长期间隔回访 → 在表达中尝试调用**

背诵不是一次打勾，而是建立可长期调用的语言材料。

### 阅读理解

基础闭环：

> **读懂大意 → 看清题目 → 回原文定位 → 找关键词 / 证据 → 组织完整回答**

随着能力提升，再逐步增加段落关系、人物和事件、情感、表达效果和篇章结构。

### 表达与写作

低龄阶段先解决“有内容可说”和“能组织清楚”，再提高文字质量：

> **看 / 读 / 经历 → 口头说清楚 → 组织 1–3 句 → 写下来 → 自己读一遍 → 修改最明显问题**

长期阶梯：

> **口头一句 → 口头三句 → 写一句 → 写三句 → 短段落 → 完整习作**

阅读和背诵中的优秀表达应逐步迁移到自己的表达中，但不要求生硬套用。

## 3. 数学：理解、数感、计算与几何

数学主线：

> **概念理解 → 数感 → 计算 → 几何与空间 → 数量关系 / 典型模型 → 问题解决 → 检查与迁移**

### 数感

重点建立：

- 数的大小和数量级；
- 数的拆分与组合；
- 数之间的关系；
- 对结果范围的预判；
- 估算和合理性意识。

目标是看到数字时能形成基本直觉，而不是只会机械运算。

### 计算

同时训练：

- **硬算**：规范、稳定完成加减乘除；
- **技巧**：识别拆分、凑整、运算律等更高效路径。

最小闭环：

> 看清数字和符号 → 判断大致范围 → 选择口算 / 竖式 / 脱式 → 规范计算 → 检查 → 判断结果是否合理

### 几何与空间

优先通过：

> **看 → 画 → 量 → 拼 → 移动 / 转化 → 说清关系**

培养图形关系、空间方位和几何直觉，不把几何简化成背公式。

### 新知识与典型模型

新概念、新算法、新模型采用：

> **成人示范（I do）→ 一起完成（We do）→ Lemon 独立完成（You do）**

先把模型教会，再要求独立使用。

### 问题解决

> **读题 → 定位问题 → 提取条件 → 建立关系 → 选择表示 / 模型 → 计算 → 检查**

只有关系不清、信息较多或连续出错时，才增加线段图、简图、表格或关系式。

### 训练方式

训练不是为了增加题量，而是为了：

- 验证概念；
- 形成熟练度；
- 暴露盲点；
- 检查迁移。

少量训练可以刻意覆盖：基础概念、基本方法、易错点、变式和综合迁移。

### 错误诊断

至少区分：

- 概念 / 知识缺失；
- 数感不足；
- 计算不熟或路径低效；
- 几何 / 空间关系不清；
- 模型不会；
- 会模型但不会调用；
- 读题 / 信息提取；
- 检查缺失；
- 困难后的任务退出。

错误只是诊断信号，不自动等于“再刷十道同类题”。

## 4. 英语：语言输入输出 + 语言规则

英语主线：

> **语音 / Phonics → 词汇 → Chunk / 固定搭配 → 句型 → 基础语言规则 → 听说读写 → 阅读理解与基础表达 → 长期积累**

### 词汇与 Chunk

一个核心词至少逐步建立：

- 听到能识别；
- 看到能读；
- 理解核心意思；
- 能放进熟悉句型；
- 需要时能正确拼写。

优先学习高频搭配和固定表达，不只记孤立中文释义。

### 句型与基础语言规则

基础规则包括：

- 人称和代词；
- be 动词和常见动词变化；
- 单复数；
- 冠词；
- 常见介词；
- 基础语序；
- 疑问和否定；
- 随教材逐步建立的基础时态意识。

教学顺序：

> **先听懂 / 读懂真实句子 → 观察相似句子 → 发现规律 → 成人简单总结 → 替换表达 → 在新场景再次使用**

不把英语变成脱离语境的语法刷题。

### 听说读写

当前优先：

> **听得懂 → 说得出 → 读得懂 → 能写基本内容**

每天保持短时、可理解、可重复的输入，并保证真实输出发生。

### 阅读理解与基础写作

阅读先训练主题和基本信息、文本定位以及熟悉词句在新语境中的理解。

写作从句型替换、1–3 句真实表达逐步推进，不急于成人化作文。

### 长期积累

积累核心词句、高频 Chunk、简短对话、适合背诵的短文和反复出现的基础规则。

闭环：

> **理解 → 跟读 / 使用 → 主动提取 → 次日复现 → 换场景使用 → 间隔回访**

## 5. 家长最小介入

先判断是哪类问题：

### 真不会

缺新知识或概念时直接教学并示范，不连续追问逼孩子猜。

### 有基础但没启动

只问一个能启动思路的问题。

### 注意中断

先恢复环境和任务，不立刻进入讲题。

### 困难后的退出

缩小到当前最小步骤，恢复进入任务后再继续。

### 明显敷衍

退回对应部分。不会可以说明，随机答案不算完成；不使用情绪性加码。

核心原则：

> **先判断是正常知识学习、能力盲点，还是状态问题，再做最小必要介入。**

## 6. 正反馈与激励

鼓励不只围绕“做对了”，更关注：

- 今天自己发现了一个错误；
- 遇到难题没有退出；
- 第一次能不看模板完成；
- 隔一天仍然记得；
- 换一道题仍能使用同样方法；
- 比上一次更快、更稳、更清楚。

正反馈应具体对应真实进步，让 Lemon 感受到：

> **认真理解和持续练习，会让我越来越会。**

---

## 7. 辅助方法及边界

- **主动提取**：先回忆，再看答案；
- **间隔复习**：当天、次日、几天后再提取；
- **轻量费曼**：用自己的话解释核心概念或词句；
- **深层转译**：让孩子外化第一反应、关系和卡点，但缺新知识时直接教；
- **舒尔特方格**：仅作视觉扫描和注意辅助，不替代真实学习；
- **学习块**：约 20–25 分钟学习 + 短休息，帮助形成清晰任务边界。

辅助方法只在解决明确目标时调用，不因为“方法先进”就永久加入主流程。
