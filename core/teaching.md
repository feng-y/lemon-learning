# Teaching Methods

> Audience：家长、家庭指导老师、AI  
> Purpose：维护可跨阶段复用的教学方法。  
> Not For：不描述 Lemon 当前状态，不记录某阶段进度，也不保存临时想法。

## 1. 跨学科教学循环：教会，再逐步放手

低龄学生的自主学习不是从“给一张任务卡”开始，而是在真实任务中逐步形成。所有学科默认采用同一条路径：

> **明确教学 / 示范 → 一起完成 → 支架下完成 → 独立完成 → 延迟验证 → 换场景迁移 → 支架退出**

关键原则：

- **缺知识就直接教**：新概念、新方法、新词义、新句型，不让孩子靠连续猜测获得；
- **有知识但不会启动才提示**：只给能推动下一步的最小问题或模板；
- **先示范思考过程**：成人把“我怎么看、怎么判断、为什么这样做”说出来，再让 Lemon 模仿；
- **模板是临时支架**：目标不是熟练填写卡片，而是最终不看卡也能完成动作；
- **一次做对不等于掌握**：隔天仍会、换一道题仍会、减少帮助后仍会，才算逐步内化。

可用五级支架观察独立程度：

| 等级 | 状态 |
| --- | --- |
| S0 | 成人示范或带着完成 |
| S1 | 成人通过关键提问才能继续 |
| S2 | Lemon 可依靠模板 / checklist 完成 |
| S3 | 基本独立，只需少量提醒或最后检查 |
| S4 | 独立完成、自检，并能在相近任务中迁移 |

支架等级不是给 Lemon 打分，而是帮助成人判断：**下一步应该教更多，还是退后一点。**

## 2. 语文：语言成长

主线：

> **字 → 词 → 句 → 篇 → 阅读 → 口头组织 → 表达 → 写作**

### 生字

普通生字走轻量闭环：

1. 课文中认读；
2. 理解课文义；
3. 确认正确笔顺；
4. 规范书写；
5. 组 1–2 个常用词；
6. 用 A/B 听写做延迟确认。

只有错误字、易混字、反复忘记字进入深度修复：先判断错因，再理解课文义，必要时补一个明显不同的常见引申义，让 Lemon 解释两个意思之间的联系。例如“潮水 → 思潮”，重点理解“思想像潮水一样一波一波涌来”，而不是堆大量词义。

### 阅读

当前基础闭环：

> **题目问什么 → 回原文定位 → 找关键词 / 证据 → 用完整的话回答**

不先猜答案。当前先训练整体感知、事件顺序、关键词和明显情感，不提前进入复杂文学赏析。

训练顺序同样遵循支架退出：成人先示范一次如何定位，再共同定位，再让 Lemon 按卡完成，最终不依赖卡片。

### 表达与写作

低龄阶段先建立“有内容可表达”和“能把话组织清楚”，再提高文字质量：

> **看 / 读 / 经历 → 口头说清楚 → 组织 1–3 句话 → 写下来 → 自己读一遍 → 只修改最明显的问题**

渐进阶梯：

> **口头一句 → 口头三句 → 写一句 → 写三句 → 短段落 → 完整习作**

“三句话表达卡”是口头组织和初期落笔的临时支架，不应固化成作文格式。

## 3. 数学：知识 / 模型 + 问题解决

数学不能全部归结为“读题”。先区分两类任务。

### A. 新知识与基本模型

> **概念 → 数学语言 → 典型表示 → 典型关系 / 方法 → 基本练习**

新概念、新算法、新模型优先采用：

> **成人示范（I do）→ 一起完成（We do）→ Lemon 独立完成（You do）**

先把模型教会，再要求独立使用模型。不要把缺知识误判为“不认真”或不断追问让孩子自己悟。

### B. 问题解决

核心闭环：

> **读题 → 定位问题 → 提取条件 → 建立关系 → 选择表示 / 模型 → 计算 → 检查**

先问：

1. 最后求什么？
2. 哪些条件与问题有关？
3. 每个数字对应什么量？
4. 条件之间是什么关系？

只有关系不清、信息较多或连续出错时，才增加线段图、简图、表格或关系式。使用最少但足够的表示。

### 计算

同时训练两类能力：

- 硬算：按规范完成加减乘除；
- 技巧：拆分、凑整、运算律等降低计算负担。

最小闭环：看清数字和符号 → 判断大致范围 → 选择口算 / 竖式 / 脱式 → 规范计算 → 用估算、逆运算或重算检查 → 判断结果是否合理。

### 错误归因

不统一归因于“粗心”。至少区分：

- 知识 / 概念缺失；
- 输入 / 读题；
- 信息提取；
- 数量关系；
- 模型选择；
- 计算；
- 书写布局；
- 检查缺失；
- 困难后的任务退出。

只修真正断点。

## 4. 英语：第二语言输入输出

核心闭环：

> **听 → 理解 → 跟读 → 认读 → 说 → 写 → 延迟提取**

四上阶段不以复杂语法体系为中心。

### 单词

一个核心词至少建立四个连接：

1. 听到能识别；
2. 看到能读；
3. 知道核心意思；
4. 能放进熟悉句型使用。

推荐：听音 → 看图 / 语境理解 → 跟读 → 看词认读 → 遮住主动回忆 → 必要书写 / A-B 听写 → 第二天再提取。

### 句型

先听懂和跟读，再理解使用情境，通过替换人物、地点、颜色、物品等关键词形成真实口头输出。先会用，再逐步理解规则。

### 听力与口语

每天保留短时、可理解、可重复的输入；每天至少有极短输出。目标不是一次说很多，而是输出真实发生。

### A/B 与延迟复习

A/B 两本按单双日交替，不承担不同职责。内容混合当天新内容和少量旧内容，错误内容动态回流，稳定内容退出高频复测。

复习优先“先回忆，再看答案”。

## 5. 家长最小介入

先判断状态，再介入：

### 真不会

缺新知识或概念时直接教学并示范，不连续追问逼孩子猜。

### 有基础但没启动

只问一个能启动思路的问题，例如“题目最后求什么？”“你第一步准备做什么？”

### 注意中断

先恢复环境和任务，不立刻进入讲题。

### 困难后的退出

缩小到当前最小步骤，例如“先不算，只说题目问什么”。恢复进入状态后再继续。

### 明显敷衍

退回对应部分。不会可以说明，随机答案不算完成；不使用情绪性加码。

核心原则：

> **先判断是知识缺口、执行卡点还是状态问题，再做最小必要介入。**

成人介入的长期方向不是“越来越会监督”，而是：

> **示范 → 支持 → 观察 → 逐步退出**

## 6. 能力成长：看支架是否退出

统一方向：

> **成人带领 → 共同执行 → 支架执行 → 独立执行 → 自检与迁移**

长期评价优先看四个维度：

1. **Independence 独立性**：完成任务需要多少外部支架？
2. **Persistence 困难停留**：遇到困难能否留在任务中并说出卡点？
3. **Quality 完成质量**：是否达到基本标准并发生自检？
4. **Retention / Transfer 保留与迁移**：隔一段时间、换相近场景还能否完成？

一次正确只是证据之一。真正的进步通常表现为：同类任务中支架从 S0/S1 逐步降到 S2/S3/S4。

## 7. 辅助方法及边界

### 主动提取 / Retrieval Practice

先回忆，再看答案。用于确认真实掌握，不把反复重读当复习。

### 间隔复习 / Spaced Repetition

当天、次日、几天后再提取。稳定内容逐步拉长间隔，反复错误重新进入高频。

### 轻量费曼

让 Lemon 用自己的话教给别人：怎么读、什么意思、什么时候用。适合英语核心词句和部分概念，不做长篇解释。

### 深层转译

让孩子外化第一反应、关系和卡点；但缺新知识时直接教学，不把“启发”变成猜答案。

### 舒尔特方格

可作为视觉扫描和持续注意的辅助训练，但不能替代真实数学读题、阅读定位或任务执行训练。

### 学习块

适龄采用约 20–25 分钟学习 + 短休息。开始前准备身体需求和材料；合理中断后回到原任务。

辅助方法只在解决具体问题时调用，不因为“方法先进”就永久加入主流程。
