# Teaching Methods

> Audience：家长、家庭指导老师、AI  
> Purpose：维护可跨阶段复用的教学方法。  
> Not For：不描述 Lemon 当前状态，不记录某阶段进度，也不保存临时想法。

## 1. 语文：语言成长

主线：

> **字 → 词 → 句 → 篇 → 阅读 → 表达 → 写作**

当前阶段方法重点：

### 生字

普通生字走轻量闭环：

1. 课文中认读；
2. 理解课文义；
3. 确认正确笔顺；
4. 规范书写；
5. 组 1–2 个常用词；
6. 用 A/B 听写做延迟确认。

只有错误字、易混字、反复忘记字进入深度修复：先判断错因，再理解课文义，必要时补一个明显不同的常见引申义，让 Lemon 解释两个意思之间的联系。默认不堆大量词义和罚抄。

### 阅读

当前核心：

> **题目问什么 → 回原文定位 → 找关键词 / 证据 → 用完整的话回答**

不会答时先回原文，不先猜。当前只训练整体感知、事件顺序和明显情感，不提前做复杂文学赏析。

### 表达与写作

渐进阶梯：

> **口头一句 → 写一句 → 三句话 → 短段落 → 完整习作**

先保证愿意落笔、围绕题目、事情说清楚、顺序基本合理，再逐步提升细节和表达质量。

## 2. 数学：信息处理

核心闭环：

> **读题 → 定位问题 → 提取条件 → 建立关系 → 选择表示 / 模型 → 计算 → 检查**

### 计算

同时训练两类能力：

- 硬算：按规范完成加减乘除；
- 技巧：拆分、凑整、运算律等降低计算负担。

最小闭环：看清数字和符号 → 判断大致范围 → 选择口算 / 竖式 / 脱式 → 规范计算 → 用估算、逆运算或重算检查 → 判断结果是否合理。

### 应用题

先问：

1. 最后求什么？
2. 哪些条件与问题有关？
3. 每个数字对应什么量？
4. 条件之间是什么关系？

只有关系不清、信息较多或连续出错时，才增加线段图、简图、表格或关系式。原则是使用最少但足够的表示。

### 错误归因

不要统一归因于“粗心”。至少区分：

- 输入 / 读题；
- 信息提取；
- 数量关系；
- 概念；
- 计算；
- 书写布局；
- 检查缺失；
- 心理退出。

只修真正断点，不对所有错题使用同一种方法。

## 3. 英语：第二语言输入输出

核心闭环：

> **听 → 理解 → 跟读 → 认读 → 说 → 写 → 复习**

四上阶段不以复杂语法体系为中心。

### 单词

一个核心词至少建立四个连接：

1. 听到能识别；
2. 看到能读；
3. 知道核心意思；
4. 能放进熟悉句型使用。

推荐：听音 → 看图 / 语境理解 → 跟读 → 看词认读 → 遮住主动回忆 → 必要书写 / A-B 听写 → 第二天再提取。

### 句型

先听懂和跟读，再理解使用情境，通过替换人物、地点、颜色、物品等关键词形成真实口头输出。先会用，再逐步理解规则。

### 听力与口语

每天保留短时、可理解、可重复的输入；每天至少有极短输出。目标不是一次说很多，而是输出真实发生。

### A/B 与延迟复习

A/B 两本按单双日交替，不承担不同职责。内容混合当天新内容和少量旧内容，错误内容动态回流，稳定内容退出高频复测。

复习优先“先回忆，再看答案”。

## 4. 家长最小介入

先判断状态，再介入：

### 真不会

缺新知识或概念时直接教学，不连续追问逼孩子猜。

### 有基础但没启动

只问一个能启动思路的问题，例如“题目最后求什么？”“你第一步准备做什么？”

### 注意中断

先恢复环境和任务，不立刻进入讲题。

### 心理退出

缩小到当前最小步骤，例如“先不算，只说题目问什么”。恢复进入状态后再继续。

### 明显敷衍

退回对应部分。不会可以说明，随机答案不算完成；不使用情绪性加码。

核心原则：

> **先判断卡在哪一层，再只做最小必要介入。**

---

## 5. 能力成长路径

统一方向：

> **外部规则 → 共同执行 → 提示下自检 → 独立执行 → 初步自我管理**

评价重点不是“这次做对没有”，而是：

- 父母提醒是否下降；
- Lemon 自己启动是否增加；
- 是否更能暴露卡点；
- 是否形成自检；
- 是否能把模板动作逐步内化。

---

## 6. 辅助方法及边界

### 主动提取 / Retrieval Practice

先回忆，再看答案。用于确认真实掌握，不把反复重读当复习。

### 间隔复习 / Spaced Repetition

当天、次日、几天后再提取。稳定内容逐步拉长间隔，反复错误重新进入高频。

### 轻量费曼

让 Lemon 用自己的话教给别人：怎么读、什么意思、什么时候用。适合英语核心词句和部分概念，不做长篇解释。

### 深层转译

不直接给答案，让孩子先外化第一反应、关系和卡点。缺新知识时仍应直接教学。

### 舒尔特方格

可作为视觉扫描和持续注意的辅助训练，但不能替代真实数学读题、阅读定位或任务执行训练。

### 学习块

适龄采用约 20–25 分钟学习 + 短休息。开始前准备身体需求和材料；合理中断后回到原任务。

辅助方法只在解决具体问题时调用，不因为“方法先进”就永久加入主流程。
