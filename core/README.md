# Core

这里保存**长期稳定、跨阶段复用**的内容。

只回答两类问题：

1. 我们长期遵循什么学习与家庭教育原则？
2. 家长、老师、AI 在不同学科和常见场景下应该怎么教、怎么介入？

Core 不记录 Lemon 当前处于什么状态，也不记录某个暑假、某一周或某一次冲突。

当前内容：

- `learning-os.md`：长期原则、家庭边界、统一学习闭环；
- `teaching/chinese.md`：语文教学方法；
- `teaching/math.md`：数学教学方法；
- `teaching/english.md`：英语教学方法；
- `teaching/parent-intervention.md`：家长最小介入；
- `teaching/capability-progression.md`：从外部规则走向独立执行的能力成长路径；
- `teaching/auxiliary-methods.md`：A/B 听写、主动提取、间隔复习、轻量费曼、舒尔特等辅助方法。

更新原则：只有当一个方法被多次真实使用并证明具有长期复用价值时，才修改 Core。
