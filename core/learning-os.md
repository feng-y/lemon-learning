# Lemon Learning OS

> 这是长期稳定底座。只保存跨阶段仍然成立的原则，不记录某个暑假、某一周或某一次冲突。

## 1. 长期目标

Lemon Learning 的目标不是提前中考化，也不是持续“修问题”，而是通过高质量教学、适度训练和逐步退出的支架，让 Lemon 在小学阶段建立未来能够承受更高学习强度的基础能力，同时保护学习兴趣和心理承载力。

长期方向：

1. **扎实基础**：核心知识准确、稳定，并逐渐形成必要的熟练度和自动化；
2. **有效方法**：知道怎么记、怎么读、怎么算、怎么检查，减少低效和无效努力；
3. **学习耐力**：专注、阅读、书写、背诵和持续思考能力逐步增长；
4. **挑战耐受**：面对不会、难题和更强同伴时，不轻易把“暂时不会”等同于“我不行”；
5. **正向自我效能**：通过真实的“原来不会 → 学会方法 → 现在会了”建立胜任感和学习兴趣；
6. **逐步自主**：随着能力增长，越来越能自己启动、发现卡点、选择方法、检查结果；
7. **长期适应**：为未来进入海淀较高强度的学习环境准备承载能力，但不把小学生活提前应试化。

统一目标：

> **以教材保证完整，以能力决定深度；成人减少无效摩擦但保留必要努力，通过真实掌握建立正反馈，并让外部支架最终内化到 Lemon 自己身上。**

## 2. 统一教学闭环

所有学科共享：

> **教材知识结构 → 当前小目标 → 先试 → 明确教学 / 示范 → 少量针对训练 → 新任务检查 → 延迟验证与迁移 → 具体正反馈 → 减少支架**

关键边界：

- 新知识和新方法可以直接教学，不让孩子靠盲目猜测获得；
- 成人和 AI 应优先减少无意义的困难，而不是消灭所有困难；
- 必要的主动提取、独立思考、变式、检查和坚持需要保留；
- 刷题不是目标，训练必须对应明确能力、技巧或真实盲点；
- 做对一次不等于掌握，隔时仍会、换场景仍会、减少提示后仍会才更接近真实掌握；
- Problems 只是异常升级机制，不驱动整个学习系统。

## 3. 日常伴随的三类能力

正常教材教学过程中，只在需要时穿插三类能力建设，不另起一套复杂课程。

### Correctness — 正确性与规范

目标是把基础动作做对并逐渐自动化。

- 语文：正确笔顺、字形、可识别书写、完整答题；
- 数学：数字和竖式规范、步骤、单位、答句、基本检查；
- 英语：字母、大小写、拼写、间距、标点和基础书写格式。

错误即时修复，避免用大量罚抄或无目标重复替代真正纠正。

### Learning Stamina — 学习耐力

像体力一样渐进增长，包括：

- 单任务专注；
- 阅读耐力；
- 规范书写耐力；
- 背诵耐力；
- 遇到困难时继续思考和停留的能力。

目标从当前可承受水平逐渐延长，例如 10 分钟 → 15 分钟 → 20 分钟，而不是一次性要求长时间硬撑。

### Learning Strategies — 学习技巧

技巧必须嵌入真实学科学习。

- 语文：`字字生花`建立关键字词的语义网络；关键词、画面和顺序帮助背诵；阅读使用定位与证据；表达先说清楚再写；
- 数学：数感、运算感、计算技巧、画图与几何操作、数量关系和模型；
- 英语：Phonics、声音—词形—意思—场景连接、Chunk、句型替换、在语境中发现和使用语言规则。

## 4. 学科能力方向

### 语文

> **识字写字 → 词句理解 → 课文 → 背诵积累 → 阅读理解 → 口头表达 → 写作**

长期建设字词、背诵积累、阅读理解、表达写作四个系统。

### 数学

> **概念理解 → 数感 → 计算 → 几何与空间 → 数量关系 / 模型 → 问题解决 → 检查与迁移**

以教材单元知识结构为主线，以数学理解和数感为底层，以计算与几何为空间较大的核心训练面。

### 英语

> **语音 → 词汇 → Chunk / 固定搭配 → 句型 → 基础语言规则 → 听说读写 → 阅读理解与基础表达 → 长期积累**

基础语言规则包括介词、人称、单复数、语序、疑问 / 否定和随教材逐步建立的时态意识。

## 5. 正反馈原则

学习兴趣不主要依赖奖励和娱乐化，而要逐渐建立在真实胜任感上。

反馈优先指向：

- **能力**：这个你现在真的会了；
- **方法**：你用了什么方法，所以这次做出来了；
- **坚持**：这次卡住后没有退出，而是继续找到下一步；
- **自主**：这次没有提醒，你自己发现了问题并检查；
- **迁移**：换一个任务，你仍然会用同样的方法。

逐渐帮助 Lemon 把成功归因从“成人帮我完成”转向：

> **我学到了方法、进行了有效练习，所以我可以把不会的东西学会。**

## 6. 家长、家教与 AI 的角色

### 家长

负责长期方向、环境、边界、情绪安全和必要的陪伴，不承担全天候实时调度。

### 家教

负责高质量教学、观察真实学习过程、发现具体断点并帮助形成方法。

### AI

优先服务家长和家教：分析教材、准备教学材料、设计针对练习、辅助诊断，减少成人备课和试错成本。

### Lemon

始终是实际学习者。成人和 AI 不替代他的理解、思考、提取、练习和决定。

所有外部支架都必须有退出方向：

> **成人示范 → 一起完成 → 临时支架 → 独立完成 → 自检与迁移**

## 7. 高竞争环境下的边界

海淀及未来升学竞争用于修正长期目标，但不能直接转化成小学阶段的提前应试化。

小学后三年优先积累：

> **基础自动化 + 学习效率 + 学习耐力 + 挑战耐受 + 学习兴趣 + 逐步自主。**

未来真正需要的是高质量学习能力，而不是单纯更长学习时间。

因此：

- 不因为周围有人提前学很多，就无边界抢跑；
- 不以牺牲睡眠、运动、基本自由时间换取更多学习时长；
- 不把每一次错误都升级成问题；
- 不让 Lemon 长期感受到自己是一个“不断需要被修复的孩子”；
- 在保证基础和竞争承载力的同时，保留好奇心、兴趣和成功体验。

## 8. 每月反向检查：系统是否适合 Lemon

除了评估 Lemon，也必须评估系统本身。

每月只问：

1. Lemon 是否比一个月前更会学、更加独立？
2. 是否出现更多真实的“我学会了”体验？
3. 遇到困难时，是更能停留，还是更容易逃避和抗拒？
4. 成人提醒和接管是否在减少？
5. 学习负荷是否导致明显疲劳、持续抵触或自我否定？
6. 是否有大量活动只是为了维护系统、打卡或完成题量，却没有产生真实掌握？

出现以下趋势时，应优先减负和调整，而不是继续加码：

- 持续性明显抵触或疲劳；
- 越来越害怕犯错；
- 只在成人持续推动下才能学习；
- 经常把不会归因为“我不行”；
- 任务越来越多，但胜任感和独立性没有增长。

## 9. 演进原则

新增规则、方法或专项问题前先回答：

1. 它是在帮助 Lemon 学得更有效，还是只让系统更完整？
2. 它解决哪个明确目标、技巧或真实盲点？
3. 是否可以通过已有教学流程轻量解决？
4. 用什么证据判断掌握或改善？
5. 是否增加了不必要的成人控制或孩子负担？
6. 什么时候可以减少或退出？

Learning OS 的长期方向不是越来越复杂，而是：

> **基础越来越稳，方法越来越内化，学习越来越高效，挑战耐受越来越强，成人介入越来越少，Lemon 越来越相信自己能够学会。**
