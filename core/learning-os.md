# Lemon Learning OS

> 这是长期稳定底座。只保存跨阶段仍然成立的原则，不记录某个暑假、某一周或某一次冲突。

## 1. 长期目标

Lemon Learning 的核心不是持续“修问题”，而是通过正常、系统、有目标的学习，让知识逐步转化为能力，并最终形成越来越强的自主学习能力。

长期方向：

1. 有完整的教材知识结构，不遗漏核心内容；
2. 在每个学科形成真正的底层能力，而不是只完成作业；
3. 通过少量、有目的的训练形成熟练度和迁移；
4. 在正常学习中及时发现薄弱点和盲点；
5. 用针对性修复代替无目的重复刷题；
6. 通过真实进步、成功体验和具体反馈建立学习正反馈；
7. 随能力增长逐步减少父母和模板支架，走向自主学习。

统一目标：

> **以教材保证完整，以能力决定深度；教学是主线，问题是反馈，训练服务于理解、掌握和迁移。**

## 2. 统一教学闭环

所有学科共享：

> **教材知识结构 → 阶段目标 → 引导理解 / 明确示范 → 针对训练 → 暴露薄弱点 → 针对修复 → 延迟验证与迁移 → 正反馈 → 减少支架**

关键边界：

- 新知识和新方法可以直接教学，不让孩子靠猜获得；
- 已经学过但不会调用时，再使用关键提问、图示或模板支架；
- 刷题不是目标，训练必须对应明确能力或盲点；
- 做对一次不等于掌握，隔时仍会、换场景仍会才更接近真实掌握；
- Problems 只处理持续阻碍正常学习的问题，不驱动整个学习系统。

## 3. 学科能力方向

### 语文

> **识字写字 → 词句理解 → 课文 → 背诵积累 → 阅读理解 → 口头表达 → 写作**

长期建设字词、背诵积累、阅读理解、表达写作四个系统。

### 数学

> **概念理解 → 数感 → 计算 → 几何与空间 → 数量关系 / 模型 → 问题解决 → 检查与迁移**

以教材单元知识结构为主线，以数学理解和数感为底层，以计算与几何为空间较大的核心训练面。

### 英语

> **语音 → 词汇 → Chunk / 固定搭配 → 句型 → 基础语言规则 → 听说读写 → 阅读理解与基础表达 → 长期积累**

基础语言规则包括介词、人称、单复数、语序、疑问 / 否定和随教材逐步建立的时态意识。

## 4. 家庭与执行原则

### 不会可以，不认真不可以

知识不会可以教学、拆解和练习；随意填写、假装完成不视为完成。

### 情绪允许，责任不能取消

可以烦躁、觉得困难，也可以短暂暂停；但不能通过发火、拖延、乱跑或乱写让任务和规则消失。

### 父母情绪会过去，任务不会自动消失

任务未达到标准时，只退回对应部分；修改合格后结束，不追加情绪性惩罚。

### 先示范，再支架，再逐步放手

> **成人示范 → 一起完成 → 临时支架 → 独立完成 → 自检与迁移**

所有外部控制都应有退出方向。

### 普通问题轻量处理，反复问题才专项修复

一次错误首先回到正常教学；只有跨时间、跨任务持续阻碍学习的问题，才进入 Problems。

### 学习与高刺激娱乐分离

学习时不播放故事、视频或其他语言内容。减少无边界、低成本、高刺激的即时娱乐，但不取消运动、亲子陪伴、正常休息和基本快乐。

## 5. 家庭底线

- 汉字书写遵循正确笔顺；
- 作业和练习清晰、完整、可识别；
- 明显乱写、漏题、无法辨认的部分不算完成；
- 网课或明确教学结束后，约 30 分钟内做短确认；
- 事实、感受、理由、责任分开；
- 公共场所不得因情绪跑离父母视线，安全优先于所有学习问题。

## 6. 家长角色

家长主要承担：

- 教学目标和学习环境维护；
- 必要的新知识示范；
- 薄弱点和盲点诊断；
- 有针对性的反馈与鼓励；
- 在能力形成后主动撤掉支架。

不承担全天候实时调度，也不成为永久答案提供器。

## 7. 演进原则

新增规则、方法或专项问题前先回答：

1. 它属于正常教材教学、能力建设，还是持续性障碍？
2. 它解决哪个明确目标或盲点？
3. 是否可以通过已有教学流程解决？
4. 用什么证据判断掌握或改善？
5. 什么时候可以减少或退出？

Learning OS 的长期方向不是越来越复杂，而是：**教材主线越来越清楚，能力结构越来越完整，专项问题越来越少，外部支架越来越轻。**
