# Lemon Learning OS

> 这是长期稳定底座。只保存跨阶段仍然成立的原则，不记录某个暑假、某一周或某一次冲突。

## 1. 长期目标

目标不是依赖父母监督获得短期成绩，而是逐步形成：

1. 能自己启动任务；
2. 能持续完成一个适龄学习块；
3. 遇到困难不立即退出；
4. 不会时能暴露真实卡点；
5. 能规范输出并对提交结果负责；
6. 能进行基本自检；
7. 能把外部规则逐渐内化为自己的学习流程。

统一目标句：

> **开始了就认真完成；不会就说明卡点；落笔就遵守规范；提交就对结果负责。**

## 2. 核心原则

### 不会可以，不认真不可以

知识不会属于能力问题，可以教学、拆解和练习；随意填写、假装完成不视为任务完成。

### 情绪允许，责任不能取消

可以失望、烦躁、觉得困难，也可以短暂暂停；但不能通过发火、拖延、乱跑或乱写让任务和规则消失。

### 父母情绪会过去，任务不会自动消失

任务未达到标准时，只退回对应部分；修改合格后结束，不追加情绪性惩罚。

### 先外部规则，再逐步放手

> 父母示范 → 父母提醒 → Lemon 自检 → 父母抽查 → Lemon 独立

所有外部控制都应有退出方向。

### 规则稳定，不因单次冲突持续增加

长期问题必须有反复事实、干预、证据和退出条件。一次冲突或一次错题不升级为永久规则。

### 学习与高刺激娱乐分离

学习时不播放故事、视频或其他语言内容。减少无边界、低成本、高刺激的即时娱乐，但不取消运动、亲子陪伴、正常休息和基本快乐。

### 普通问题轻量处理，反复问题才深度修复

普通生字、普通计算、熟练任务走最短流程；错误字、反复错题、持续性问题才调用专项方法。

## 3. 家庭底线

- 汉字书写遵循正确笔顺；
- 作业和练习清晰、完整、可识别；
- 明显乱写、漏题、无法辨认的部分不算完成；
- 网课或明确教学结束后，约 30 分钟内做短确认；
- 事实、感受、理由、责任分开；
- 公共场所不得因情绪跑离父母视线，安全优先于所有学习问题。

## 4. 统一学习闭环

三科学习共享：

> **输入 → 理解 → 输出 → 检查 → 反馈 → 延迟确认**

具体表现：

- 语文：语言理解与表达；
- 数学：信息处理、关系建立与验证；
- 英语：第二语言输入、理解、输出与重复。

## 5. 家长角色

家长主要承担：

- 规则维护者；
- 任务教练；
- 断点诊断者；
- 具体反馈提供者。

不承担全天候实时调度，也不成为答案提供器。

## 6. 演进原则

新增方法或规则前先回答：

1. 它解决哪个反复出现的真实问题？
2. 它属于长期原则、教学方法、当前阶段策略还是临时想法？
3. 用什么现象判断有效？
4. 什么情况下应该退出？

Learning OS 的长期方向不是越来越复杂，而是随着 Lemon 成长，逐步减少外部控制。
